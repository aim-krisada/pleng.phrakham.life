# สรุป 1 หน้า — คืนพื้นที่จอหน้าแก้เพลง (เคาะทิศ · ให้ P'Aim)

**ปัญหา:** แถบเครื่องมือด้านล่างกินพื้นที่จอ → เนื้อเพลงเหลือน้อย พี่เปาทำงานยาก โดยเฉพาะมือถือ/จอพับ

**ทางแก้ (ให้แถบ "หลบเอง" ไม่ใช่ให้คนย่อ):**
1. **เลื่อนดูเพลง → แถบหลบหายเอง** (จอโล่ง เห็นเนื้อเพลงเต็ม) · เลื่อนขึ้น/แตะปุ่มเล็ก → กลับมา
2. **แตะโน้ต/ห้อง/บรรทัด → เครื่องมือโผล่ตรงนั้น** (ไม่ต้องมีแถบใหญ่ค้างตลอด)
3. **พิมพ์ → แถบซ่อน เครื่องมือเกาะเหนือแป้นพิมพ์** (ไม่โดนแป้นบัง)
4. **แถบเริ่มต้นเหลือปุ่มใช้บ่อยจริง** ที่เหลือเก็บใน ⚙ (พี่เปาปรับ/ปักกลับเองได้)

**ได้ทุกจอ:** มือถือ · จอพับ (พับ/กาง) · tablet · คอม · คนตาบอด (แถบไม่หลบ) · ของที่ทำค้างไม่หายตอนพับ-กาง-หมุนจอ

**พี่เอมจะได้:** หน้าแก้เพลงจอโล่ง เนื้อเพลงเด่น เครื่องมือมาเมื่อต้องใช้ — เหมือน Google Docs บนมือถือ

**ผลพลอยได้:** แถบนี้ใช้ร่วมกับเว็บพระคำ → พระคำได้แถบอ่านออกเสียงที่หลบเองด้วย (แต่ต้องทดสอบ 2 เว็บ)

**สร้างยังไง:** บนโค้ดจริง เป็นเฟส A→D (แถบ slim+หลบ → เครื่องมือตามการแตะ → รู้จักแป้นพิมพ์+ทุกจอ) · tester ตรวจครบ · **พี่เอมลองแอปจริงครบชิ้นเดียวจบ** ไม่มี demo ทิ้ง

**เคาะทิศนี้ไหม → สร้างเลย** (ปรับได้บนของจริงระหว่างทาง ไม่รื้อ)
