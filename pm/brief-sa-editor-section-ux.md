# Brief — SA: ทำโหมดแก้ไข (จัดการท่อน/ข้อ) ให้ intuitive ระดับมืออาชีพสากล

**สั่งโดย:** pm4 (จาก P'Aim 10 ก.ค.) · **docs only — ⛔ ไม่แตะ code** · **branch:** `git switch -c sa-editor-section-ux studio-shell-redesign`

## ปัญหาจริง (ตั้งต้นจากตรงนี้ ไม่ใช่จากไอเดียแก้)
**พี่เปา (นักดนตรี · ไม่ใช่สายเทคนิค) เปิดโหมด "แก้ไข" แล้ว "ไม่รู้ต้องทำยังไง"** เพื่อ แก้ชื่อท่อน · จัดลำดับข้อ · แก้เนื้อร้องที่มีอยู่ · แก้เพลงที่ import มาแล้วข้อสลับ → **แปลว่า design ปัจจุบันไม่ intuitive พอ**
P'Aim ต้องการให้ **"ดูเป็นเว็บมืออาชีพ ใช้งานระดับสากล เหมือนเว็บระดับ high ทั่วโลก"** — เปิดมาแล้วรู้เองว่ากดตรงไหน โดยไม่ต้องมีคนสอน

## งาน = UX audit + ข้อเสนอปรับปรุง (ไม่ใช่สั่งทำ feature ตายตัว)
1. **ศึกษาของที่มีอยู่จริง** — `src/components/EditorMode.vue` โหมดแก้ไข โดยเฉพาะ:
   - โมเดล v2: **ทำนอง (stanza)** = โน้ตล้วน · **ลำดับเพลง/ข้อ (arrangement)** = เลือกทำนอง+ใส่เนื้อ+ชื่อ+คีย์
   - **sidebar "ส่วนของเพลง"** (rail): กลุ่ม ทำนอง / เนื้อร้อง / ขั้นสูง → ปุ่ม "📜 ลำดับเพลง"
   - **บล็อก "📜 ลำดับเพลง"** (ล่างสุด): แต่ละแถว = [ท่อนทำนอง ▾][ช่อง "ชื่อข้อ"][คีย์][▲▼ ✎ ✕]
   - แก้เนื้อ: กล่องรายพยางค์ใต้โน้ต (เลือกข้อด้วย ✎) + แผง "📝 แก้เนื้อแบบย่อหน้า"
   - ลองใช้จริง (LAN/live) มองผ่านสายตา "คนไม่เทคนิคเปิดครั้งแรก"
2. **วินิจฉัยว่าทำไมงง** — จุดที่ไม่ intuitive (เช่น ฟีเจอร์ซ่อนในบล็อกล่าง · แก้ชื่อ 1 ที่ แต่ชื่อโชว์หลายที่ · คำว่า "ขั้นสูง" · แก้เนื้อ 2 ที่ · ไม่มี affordance ว่าคลิก/ลากได้ · มือถือหาไม่เจอ ฯลฯ) — เขียนเป็นรายการปัญหา + เหตุผล
3. **เทียบมาตรฐานสากล** — ดูว่าเว็บ/แอปมืออาชีพระดับโลกจัดการ "แก้ชื่อ inline · จัดลำดับรายการ · แก้เนื้อหาในที่" ยังไง (inline edit, drag-to-reorder ที่รองรับ touch, progressive disclosure, direct manipulation) · อ้าง WCAG 2.2 AA + responsive (memory `pleng-ui-sop`) · อ่าน `des-sop` ถ้ามี
4. **เสนอการปรับปรุง** — ทางที่ทำให้ผู้ใช้ใหม่รู้ทันทีว่าทำอะไรได้ · **มือถือ + PC ต้องดีทั้งคู่** (drag-drop ต้องเวิร์กบน touch ด้วย)

## ไอเดียจาก P'Aim (พิจารณา · ไม่บังคับ · SA ประเมิน+เสนอที่ดีที่สุด)
- แก้ชื่อ **in-line**: คลิกที่ชื่อท่อน (ทั้งใน sidebar และที่หัวข้อท่อน) แล้วพิมพ์แก้ตรงนั้น
- จัดลำดับด้วย **ลากวาง (drag-drop)** แทนปุ่ม ▲▼
- ถ้าออกแบบดีพอ **บล็อก "ลำดับเพลง" ล่างอาจไม่ต้องมี** — หรือแทนด้วย: ปุ่มยุบย่อท่อนทั้งหมดแล้วลากวาง · หรือ ปุ่มเลื่อนขึ้น/ลง อยู่ตรงท่อนที่กำลังแก้เลย
