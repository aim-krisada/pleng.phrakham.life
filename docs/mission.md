# Mission — เพลง.พระคำ.ชีวิต (pleng.phrakham.life)

## พันธกิจ (เราทำเว็บนี้ไปเพื่ออะไร)
ให้คริสตจักรไทยมี **คลังเพลงนมัสการที่ใช้ฟรี** — รวมเนื้อร้อง คอร์ดกีตาร์ และโน้ตตัวเลข (jianpu) ไว้ที่เดียว
เพื่อให้ทุกคน ตั้งแต่คนร้องธรรมดาจนถึงคนแต่งเพลง ใช้เพลงเดียวกันได้ **ง่าย · ถูกต้อง · เป็นระเบียบ**

## ผู้ใช้ 3 กลุ่ม (tiers)
| กลุ่ม | เป็นใคร | ทำอะไรได้ |
|---|---|---|
| **Tier 0 — ไม่ล็อกอิน** | คนทั่วไป + คนทำเพลงภายนอก | ดู · ร้อง/คาราโอเกะ · พิมพ์แผ่นเพลง · upload/download JSON ของตัวเอง — **เข้าคลังไม่ได้** (ขออนุมัติทางอีเมล) |
| **Tier 1 — ผู้ทำเพลง (editor)** | ทีม คจ. ที่ได้รับเชิญ | ทุกอย่างของ Tier 0 + **เก็บในระบบ** (ร่าง / ส่งตรวจ) |
| **Tier 2 — ผู้อนุมัติ (approver)** | ทีม คจ. ที่ได้รับเชิญ | ทุกอย่างของ Tier 1 + **อนุมัติเข้าคลัง / ลบ / ย้อน** |

- เข้าระบบแบบ **เชิญเท่านั้น (invite-only)** — สมัครเองไม่ได้
- **เส้นแบ่งสิทธิ์อยู่ที่ "เก็บเข้าระบบได้ไหม" ไม่ใช่ "แก้ได้ไหม"** — ทุกคนแก้เพลงได้ ต่างกันแค่บันทึกไว้ที่ไหน (ไฟล์ตัวเอง vs คลังกลาง)

## วัดความสำเร็จยังไง
- คนทั่วไปเปิดเพลง → ร้อง / ฝึก / พิมพ์ ได้เองโดยไม่ต้องมีคนสอน
- คนทำเพลงของ คจ. → เพิ่มเพลงเข้าคลังอย่างเป็นระบบ ค้นเจอง่าย เลขไม่ซ้ำในหมวด
- คนทำเพลงภายนอก → ใช้เครื่องมือทำเพลงได้เต็มที่ผ่าน JSON ของตัวเอง โดยไม่ปนคลังหลัก
- เพลงที่บันทึก = **jianpu มาตรฐานที่นักดนตรีคนไหนก็อ่าน / พิมพ์ / ส่งต่อได้**

## หลักการทำงาน
- ฟรี + เข้าถึงง่าย · mobile-first · ไม่ล็อกอินก็ใช้ได้
- **Google-Docs-clean** — โชว์ของที่ใช้บ่อย ซ่อนของตั้งครั้งเดียวไว้ในเมนู
- **SSOT = jianpu มาตรฐานครบ** เครื่องมือช่วยจัด (ลาก / บล็อก) เป็นแค่ตัวช่วย ไม่ใช่ความจริงอีกชุด
- **ISO/IEC 29110-5-4 แบบเบา** — ทุกงานสาวกลับหา mission ได้ตามสาย `backlog → user story (US) → design spec (DS) → code` แต่ไม่เพิ่มเอกสารเยิ่นเย้อ
