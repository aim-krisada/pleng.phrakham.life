# ps2 design decisions — โหมดแก้ไข (③) + shell/เมนู (②)

Prototype กดได้จริง: `docs/design/ps2-studio-prototype.html` (เปิดใน Chromium) · เคาะโดย P'Aim ระหว่าง ps2
สถานะ: **② + ③ เคาะแล้ว** (commit นี้) · **④ ฝึกร้อง ยังต้องขัดต่อ**

## หลักที่ยึด
- Google-Docs-clean: โชว์ของใช้บ่อย · ซ่อนของนานๆใช้ · auto-save ไม่มีปุ่ม Save
- เส้นแบ่งสิทธิ์ = "เก็บเข้าระบบได้ไหม" (mission) · ทุกคนแก้ได้
- UI ลดรูปได้ แต่ **model ห้ามลดรูป** (เก็บละเอียดเผื่ออนาคต)

## ② shell / เมนู / IA
- **แถบบนร่วมทุกโหมด:** `[เพลง.พระคำ.ชีวิต ▾] · [ชื่อเพลง ▾] · [เพลง ▾] · (ฝึกร้อง | แผ่นเพลง | แก้ไข) · [เข้าสู่ระบบ]`
- **เมนูชื่อเว็บ (▾)** = ไปหน้า: รายการเพลง · คู่มือ · เกี่ยวกับเรา · พระคำ.ชีวิต ↗ (ไอคอน = **โลโก้จริง** favicon) — ตัด "ทำเพลง" (B007)
- **ชื่อเพลง:** คลิกตัวชื่อ = พิมพ์แก้ inline · `▾` = เมนูเพลงนี้ (ตั้งค่าเพลง · ออกจากเพลงนี้ · **ลบเพลง** [ผู้อนุมัติ + confirm])
- **"เพลง ▾"** (แทน "เปิดเพลง" เดิม — B008/B003): กดทีเดียวได้ทั้ง **＋สร้างเพลงใหม่ (บนสุด)** + ช่องค้นหา + รายการเปิด · จิ้ม=เปิดเลย (ไม่มีปุ่ม OK) · Esc/คลิกนอก=ปิด · **มือถือ = เต็มจอ**
- **มือถือ 2 แถว** (B009/B014/B017): (1) ชื่อเว็บ▾ + ชื่อเพลง▾ + 👤 · (2) 📄เพลง + โหมด**ไอคอนล้วน** (ไมค์/แผ่น/ดินสอ) · **ไม่มี hamburger** (▾ บอกว่าเป็นเมนูพอ)
- **ไอคอน (Lucide):** รายการเพลง=`list-music` · คู่มือ=`book-open` · เพลง=`file-music` · แผ่นเพลง+ดูผล=`music` (ตัวโน้ต) · ท่อนฮุก=`hook` (ตะขอ = pun "ฮุก") · พระคำ.ชีวิต=โลโก้จริง

## ③ โต๊ะแก้ไข
### เค้าโครง
- `.studio` = **left rail "องค์ประกอบเพลง" + content ขวา** · **max-width 1240**
- Rail = **sticky ค้างตลอด (desktop)** / **drawer สไลด์ (mobile)** · toggle ด้วย `◱` (square-chevron-left) ที่หัวโต๊ะ
- **Navigation = side rail อย่างเดียว** (มาตรฐาน master-detail; ดีกว่า dropdown สำหรับเพลงยาว + คนไม่เก่งเทคโนโลยี) · หัวโต๊ะเหลือ **breadcrumb `ท่อน A · ข้อ 1`** บอกตำแหน่ง + กดเปิด rail · **ไม่มี dropdown ซ้ำ**

### Rail (องค์ประกอบเพลง)
- **ทำนอง (ท่อน)** icon `music-2` · **เนื้อร้อง (ข้อ)** icon `text-initial` · **ลำดับการร้อง** · แต่ละกลุ่มมี ＋เพิ่ม
- เลือก ท่อน/ข้อ → **เนื้อหากลางเปลี่ยน + ไฮไลต์ sync** (2 ทาง) · สะท้อน v2: ทำนองแก้ครั้งเดียว หลายข้อใช้ร่วม

### หัวโต๊ะ (ระดับบรรทัด/ท่อน)
`breadcrumb · (i)วิธีใช้ · ?สัญลักษณ์ · [grow] · เค้าโครง(1ห้อง/แถว ⇄ ต่อกัน) · ดูผลทั้งเพลง · ▶ฟัง · [โครงเพลงด่วน: ฮุก·ซ้ำ·คัดลอก·🗑ลบ] · ⋯`
- **โครงเพลงด่วน (desktop)** = ท่อนฮุก · ซ้ำ · คัดลอกบรรทัด · **ลบบรรทัด (แดง + confirm)** · **⋯** = Fine/D.C. (นานๆใช้) · **มือถือ** = ยุบทุกอย่างใน ⋯
- **โครงเพลงทั้งหมด = ระดับบรรทัด/ท่อน — ไม่มีราย ห้อง** (แต่ละ ห้อง เหลือแค่ ▶ฟัง + ดูผล)
  - **ผูกไว้ใน DS:** UI ระดับบรรทัด · **model เก็บ repeat/repeatEnd/volta/end ราย ห้อง คงเดิม (v2 ps1)** → future-proof: อยากได้ mid-line repeat วันหน้า = เพิ่มปุ่ม UI ไม่ต้องแปลงข้อมูล

### ช่องโน้ต — 4 ชั้นต่อช่อง (แบบ ก: พิมพ์โค้ด/เห็นผลสด)
- **คอร์ด (picker ด้านบน) · jianpu render สด · ช่องโค้ด (พิมพ์ `.5` `6_` `6~`) · ช่องคำ**
- แต่ละ **ห้อง** มี ▶ฟัง + **ดูผล (toggle แก้ ⇄ แผ่นเพลง — แทนที่ ไม่ซ้อนแถว)** · **ดูผลทั้งเพลง** = สั่งทุกห้องพร้อมกัน (ป้ายสะท้อนสถานะจริง)
- jianpu render จริง: จุดบน/ล่าง · ขีดใต้(เขบ็ต) · จุดขวา(เพิ่มครึ่ง) · `–`(ลาก) · เส้นโค้ง tie (เอื้อน)

### แก้เนื้อร้อง (B005) — 2 ที่ ไม่ซ้ำ
- **รายพยางค์** (ช่องคำใต้โน้ต — แม่น) + **พาเนล "พิมพ์เนื้อทั้งข้อ"** (ยุบ/กาง — พิมพ์เร็ว) · **ตัด** textarea ในหน้า "ลำดับเพลง"

### Save / help / dock
- **ไม่มีปุ่ม Save** = **auto-save** + **"✓ บันทึกร่างแล้ว" (sticky มุมขวาบน ค้างตลอด)**
- **ดาวน์โหลด JSON / ส่งตรวจ / อนุมัติ** = อยู่ใน **dock** (settled ps1 · ไม่แสดงใน prototype) · ตาม tier
- **ไม่มี dock ใน prototype** (คีย์โน้ต = ของเสร็จ ps1) · **legend "?"** ย้ายมาหัวโต๊ะ
- **Help 3 ชั้น:** UI ตัวเอง (พิมพ์เห็นเลย) → (i)วิธีใช้ / ?สัญลักษณ์ (in-context) → **คู่มือฉบับเต็ม** (ลิงก์)

## ④ ฝึกร้อง (viewer) — เคาะแล้ว (prototype v36)
- **B016** จอเลื่อนตามโน้ต · ปั่นสกอลล์เอง = auto-scroll พัก 3.5s · **ปุ่มหยุด sticky ใน dock ตามติดจอ**
- **B006** ไฮไลต์ราย**โน้ต+คำ** (ไม่ใช่ทั้ง segment) · **แตะบรรทัด/โน้ต = กระโดดร้องจากตรงนั้น** · ผูก v2 (worktree แยก · model 1 พยางค์/โน้ต)
- **ห้องไหลต่อกันเป็นบรรทัด** (เส้นแบ่งห้อง `|` · พับตามความกว้าง) — ใช้กับ **แผ่นเพลง** ด้วย · ไม่แยกท่อนสั้นๆ ทีละแถว
- **ชิปกระโดดท่อน** (ร้อง 1 · รับ · ร้อง 2) ด้านบน
- **Control bar (dock) = แบบแอปเดิม (B024):** แสดงผล ▾ · คอร์ด ▾ · **คีย์ [E (ต้นฉบับ)] ▾** · **ความเร็ว = ศัพท์ดนตรี** (Grave/Largo/Larghetto/Adagio/Andante/Moderato/Allegro/Vivace/Presto + ♩=BPM + คำไทย) · 🔁 **วนซ้ำ multi-select ท่อน/ข้อ** · ก− ก+
- **Mobile: ไม่ scroll แนวนอน → กด "เพิ่มเติม" ขยายขึ้นบน (expand-up)** · default = ▶ + คอร์ด + ความเร็ว
- **ค้าง (B024 → ps3):** customizable dock — reuse "deck-key" library ของหน้าแก้ไข (ให้คนเลือกปุ่มเอง · แบบเลื่อน/ขยายขึ้น)
