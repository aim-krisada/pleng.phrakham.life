Remix + ล็อกสูตรการบรรเลง — wireframe

คลิก 🎲 สุ่มสูตรใหม่ เพื่อดูสูตรเปลี่ยน (แบบที่ N) · คลิก ล็อก เพื่อปัก · ป้าย + caption ของ MP3 อัปเดตพร้อมกัน (ยืนยันว่า "ไฟล์ = ที่ฟัง"). นี่คือภาพร่างเพื่อเคาะทิศทาง — ไม่ใช่โค้ดจริง.

Remix อยู่ในป็อปอัพเสียงดนตรี (ไม่เพิ่มปุ่มบนแถบ = คงหลัก "1 ปุ่มเสียง" + ไม่ล้นมือถือ). โผล่เฉพาะสไตล์บรรเลง/สงบ.

ดาวน์โหลด — MP3 ตรงกับที่ฟัง

ไฟล์ = สูตรที่กำลังฟัง · แบบเริ่มต้น

caption 1 บรรทัดบอกว่าไฟล์ที่จะได้ = สูตรปัจจุบัน (ตัด doubt). เบื้องหลัง MP3 render ผ่าน arrange() ด้วย seed + สไตล์ + เทคนิคเดียวกับที่ฟัง (pass 0).

โมเดล (ย่อ)

  • สูตร (variant) = สิ่งที่ Remix เปลี่ยน + ล็อกได้ · เริ่ม "แบบเริ่มต้น" (variant 0 = เสียงเดิมของเพลงวันนี้) → กด = แบบที่ 1, 2, 3…
  • ล็อก = จำสูตรนี้ต่อเพลง (เปิดใหม่ได้เหมือนเดิม) · ไม่ล็อก = เปิดใหม่กลับเป็นแบบเริ่มต้น · กดสุ่มใหม่ = ปลดล็อกให้เอง
  • MP3 ใช้สูตร+สไตล์+เทคนิคปัจจุบัน (pass 0) → ตรงกับรอบแรกที่ได้ยินเป๊ะ · ดาวน์โหลดกี่ครั้งก็เหมือนเดิม
  • ล็อกเฉพาะ seed — ไม่ล็อก config 12 เทคนิค (มันเห็นผลสด + จำ per-browser อยู่แล้ว)
  • a11y: Tab ถึงปุ่มได้ · โฟกัสคงที่ปุ่มสุ่มหลังกด (กดรัวได้) · aria-pressed บนปุ่มล็อก · ป้าย aria-live
