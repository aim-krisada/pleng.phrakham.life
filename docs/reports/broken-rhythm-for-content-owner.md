## ปัญหานี้คืออะไร (อธิบายก่อน)

โน้ตเก็บเป็น **ข้อความ** และแต่ละตัวมีความยาวของมัน:

| เขียนแบบนี้ | ยาวเท่าไร |
|---|---|
| `5` เลขเปล่า | **1 จังหวะ** |
| `5_` ขีดใต้ 1 ขีด (เขบ็ต 1 ชั้น) | **ครึ่งจังหวะ** |
| `5__` ขีดใต้ 2 ขีด (เขบ็ต 2 ชั้น) | **หนึ่งในสี่จังหวะ** |
| `5.` จุดหลังตัวเลข (จุดเพิ่มค่า) | **ยาวขึ้นครึ่งหนึ่งของตัวเอง** (1 → 1.5) |
| `5_.` ขีดเดียว + จุด | ครึ่ง + ครึ่งของครึ่ง = **0.75 จังหวะ** |
| `-` ขีดยาว | **เพิ่มอีก 1 จังหวะ** ให้ตัวหน้า |
| `0` | ตัวหยุด (เงียบ) ยาวตามที่เขียน |

**เอาทุกตัวในห้องมาบวกกัน ต้องได้เท่ากับที่หัวเพลงบอก** — เพลง 4/4 ห้องละ 4 จังหวะ ·
3/4 ห้องละ 3 · **6/8 ห้องละ 3 จังหวะ** (ไม่ใช่ 6 — เพราะนับเป็นตัวเขบ็ต)

ถ้าบวกแล้วไม่ลงตัว เวลากดฟัง **ห้องนั้นจะยาว/สั้นกว่าที่ควร แล้วทุกอย่างหลังจากนั้นจะเลื่อนตามไปหมด**

# ★ อาการที่ 1 — เขบ็ตชั้นเดียวที่ควรเป็น 2 ชั้น (1 เพลง แต่กระทบ 40 ห้อง)

**เพลงเดียวในใบนี้ที่ผิดทั้งเพลง — แก้ก่อนเพื่อนเลย**

### 1.1 · เพลง #108 "ตกอยู่บาบูโลน" (4/4 → ห้องละ 4 จังหวะ)

| | |
|---|---|
| **อยู่ตรงไหน** | ทำนอง **ท่อน A** — ท่อนเดียวของเพลงนี้ · **ใช้ร่วมกันทั้ง ร้อง 1-5** |
| **กระทบกี่ห้อง** | **40 จาก 80 ห้อง** ทุกห้องเกินเท่ากันหมด (+0.25) |
| **แบบที่ผิดซ้ำๆ** | ห้องที่ลงท้ายด้วยคู่ `เลข_. เลข_` |

**ตัวอย่างจริง 3 ห้องแรก (บรรทัดที่ 1 จาก 4):**

| ห้อง | ตอนนี้พิมพ์ไว้ | ได้กี่จังหวะ | ตัวที่ผิด | ควรพิมพ์ว่า |
|---|---|---|---|---|
| 1 | `3 3 3_. 2_ 1` | **4.25** (ควรได้ 4) | `2_` | **`2__`** |
| 2 | `2 2 2_. 1_ .6` | **4.25** | `1_` | **`1__`** |
| 3 | `.5 .5 1_. .7_ 1_ 2_` | **4.25** | `.7_` | **`.7__`** |

**ทำไมระบบถึงนับได้เท่านั้น:**
คู่ "จุดเพิ่มค่า + ตัวสั้น" (`3_.` แล้วตามด้วยตัวถัดไป) เป็นคู่มาตรฐานที่ต้องรวมกันได้ **1 จังหวะพอดี**
`3_.` = 0.75 จังหวะ → ตัวที่ตามมาต้องเป็น **0.25** คือเขบ็ต **2 ชั้น** (`2__`)
แต่ในเพลงนี้เขียนเป็นเขบ็ต **1 ชั้น** (`2_` = 0.5) → **เกินไปห้องละ 0.25 จังหวะ**

> ✅ **มั่นใจสูง** — เมื่อเปลี่ยน `_` เป็น `__` เฉพาะตัวที่ตามหลัง `_.` **ห้องจะเต็มพอดี 4 จังหวะ**ทุกห้อง
> ⚠️ **กระทบทุกข้อ** — ท่อน A ใช้ร่วมกันทั้ง ร้อง 1-5 แก้ครั้งเดียวเปลี่ยนหมดทุกข้อ (ซึ่งเป็นสิ่งที่ต้องการ)
> 💡 **ทั้ง 40 ห้องเป็นแบบเดียวกัน** — ไล่หาทุกที่ที่มี `_.` แล้วดูตัวถัดไป ถ้าเป็น `_` ให้เปลี่ยนเป็น `__`

# อาการที่ 2 — ห้องที่จังหวะไม่ลงตัว ยังไม่แน่ใจว่าที่ถูกคืออะไร (3 เพลง)

**⚠️ กลุ่มนี้ผมไม่ฟันธงให้** — นับได้ว่าไม่ลงตัวแน่นอน แต่ **"ที่ถูกควรเป็นอะไร" มีได้หลายทาง**
เสนอเป็นทางเลือกพร้อมบอกว่าแต่ละทางจังหวะลงเท่ากันไหม · **พี่เปาเลือกจากที่ร้องจริง**

### 2.1 · เพลง #89 "ทรงเป็นกำลังยามข้าอ่อนล้า" (4/4 → ห้องละ 4 จังหวะ)

| | |
|---|---|
| **อยู่ตรงไหน** | ทำนอง **ท่อน B (ท่อนรับ)** · **บรรทัดที่ 1 จาก 2** · **ห้องสุดท้าย (ห้องที่ 4 จาก 4)** |
| **ตอนนี้พิมพ์ไว้** | `(3~ ~3_ 4_ 3_  2) 0` |
| **ได้กี่จังหวะ** | **4.5** — **เกินไป 0.5** (ควรได้ 4) |
| **ทำไมถึงเกิน** | 3(1) + ~3\_(0.5) + 4\_(0.5) + 3\_(0.5) + 2(1) + 0(1) = 4.5 |

**ทางเลือก:**
- **ทาง ก** — ตัวหยุด `0` ท้ายห้องเป็น `0_` (ครึ่งจังหวะ) → **ได้ 4 พอดี** ✅
- **ทาง ข** — `2` เป็น `2_` (ครึ่งจังหวะ) → **ได้ 4 พอดี** ✅
- ทั้งสองทางจังหวะลงเท่ากัน **ต่างกันที่เสียงไหนสั้นลง** — ต้องฟังว่าตอนร้องจริงเสียงไหนสั้น

### 2.2 · เพลง #760 "ตั้งแต่จากกันบนเขามะกอกเทศปีนั้น" (6/8 → **ห้องละ 3 จังหวะ**)

| | |
|---|---|
| **อยู่ตรงไหน** | ทำนอง **ท่อน B (ท่อนรับ)** · **บรรทัดที่ 2 จาก 2** · **ห้องสุดท้าย (ห้องที่ 4 จาก 4)** |
| **ตอนนี้พิมพ์ไว้** | `(1. 1)` |
| **ได้กี่จังหวะ** | **2.5** — **ขาดไป 0.5** (ควรได้ 3) |
| **ทำไมถึงขาด** | 1.(1.5) + 1(1) = 2.5 |

**ทางเลือก:**
- **ทาง ก** — ตัวหลังเป็น `1.` ด้วย → `(1. 1.)` = 1.5+1.5 → **ได้ 3 พอดี** ✅
- **ทาง ข** — เติมตัวหยุดครึ่งจังหวะท้ายห้อง → `(1. 1) 0_` → **ได้ 3 พอดี** ✅
- ทาง ก = ลากเสียงยาวจบห้อง · ทาง ข = จบแล้วเว้นวรรคหายใจ **ต้องฟังว่าร้องแบบไหน**
- ⚠️ เพลงนี้เป็น **6/8 ห้องละ 3 จังหวะ** (ไม่ใช่ 6) อย่าเผลอนับเป็น 6

### 2.3 · เพลง #100 "ขอสรรเสริญพระเจ้าโดยไม่หยุดยั้ง" (4/4 → ห้องละ 4 จังหวะ)

| | |
|---|---|
| **อยู่ตรงไหน** | ทำนอง **ท่อน A** (ใช้ร่วมกันทั้ง ร้อง 1-3) · **รอยต่อ บรรทัดที่ 2 → บรรทัดที่ 3** |
| **ตอนนี้พิมพ์ไว้** | ท้ายบรรทัด 2 (ห้องที่ 4) = `3 2 1_` ได้ **2.5** · ต้นบรรทัด 3 (ห้องที่ 1) = `3_ 3_. 3_` ได้ **1.75** |
| **ได้กี่จังหวะ** | ห้องนี้คร่อม 2 บรรทัด รวมกันได้ **4.25** — **เกินไป 0.25** |
| **ทำไมถึงเกิน** | ปกติห้องที่ตัดข้ามบรรทัดต้องรวมกันได้ 4 พอดี · อันนี้เกิน 1 ใน 4 จังหวะ |

**ทางเลือก:**
- **ทาง ก** — `3_ 3_. 3_` ตัวสุดท้ายเป็น 2 ชั้น → `3_ 3_. 3__` = 0.5+0.75+0.25 = **1.5** → รวม **4 พอดี** ✅ *(แบบเดียวกับ #108 เลย)*
- **ทาง ข** — `3 2 1_` ตัว `1_` เป็น `1__` → 2.25 → รวม **4 พอดี** ✅
- **ผมเอนไปทาง ก** เพราะรูป `_.` ตามด้วย `_` เป็นแบบเดียวกับที่ผิดใน #108 เป๊ะ **แต่ไม่ฟันธง**
- ⚠️ **กระทบทุกข้อ** — ท่อน A ใช้ร่วมกันทั้ง ร้อง 1-3 · ในเพลงนี้ยังมีจุดคล้ายกันอีกที่ **บรรทัด 4 ท่อน B (ท่อนรับ) ห้องสุดท้าย** = `~1 - 0_` ได้ 2.5 ขาด 1.5 → ตรวจพร้อมกันได้

# อาการที่ 3 — ท่อนใหม่เริ่มด้วยห้องสั้น แต่ไม่ได้บอกระบบว่าเป็น "ห้องยก" (1 เพลง)

**นี่ไม่ใช่โน้ตผิด** — โน้ตถูกแล้ว แต่ระบบไม่รู้ว่าห้องนั้น**ตั้งใจให้สั้น**

### 3.1 · เพลง #120 "ทรงเป็นพันธุ์เมล็ดชีวิต" (4/4 → ห้องละ 4 จังหวะ)

| | |
|---|---|
| **อยู่ตรงไหน** | ทำนอง **ท่อน B (ท่อนรับ)** · **บรรทัดที่ 1 จาก 2** · **ห้องแรก** |
| **ตอนนี้พิมพ์ไว้** | `.5` (ตัวเดียว) |
| **ได้กี่จังหวะ** | **1** จาก 4 |
| **นี่ผิดจริงไหม** | **น่าจะไม่ผิด** — เป็น **ห้องยก** (ตัวขึ้นต้นก่อนเข้าห้องเต็มห้องแรก) ซึ่งถูกต้องตามเพลง |
| **ปัญหาคือ** | ในระบบยัง**ไม่ได้ทำเครื่องหมายว่าเป็นห้องยก** ระบบจึงคิดว่าห้องนี้จังหวะขาด |

**ทางเลือก:**

- **ทาง ข** — ถ้าจริงๆ แล้วท่อนรับควรเริ่มเต็มห้อง ต้องเติมโน้ตให้ครบ 4 จังหวะ
- **ผมเอนไปทาง ก** เพราะเป็นห้องแรกของท่อนรับพอดี ซึ่งเป็นที่ที่ห้องยกอยู่ตามปกติ **แต่พี่เปารู้ดีกว่าว่าร้องยังไง**
- ⚠️ **กระทบทุกข้อ** — ท่อน B ร้องซ้ำทุกข้อ

# ที่ยังตรวจไม่จบ (ต้องรอรอบสอง)

**บอกตรงๆ:** เครื่องมือตรวจของผม 2 ตัวให้ผลไม่ตรงกัน — ตัวหนึ่งได้ **12 เพลง** อีกตัวได้ **15 เพลง**
แต่ **ทับกันแค่ 5 เพลง** ซึ่งคือ 5 เพลงในใบนี้ ส่วนที่เหลือ **ยังบอกไม่ได้ว่าตัวไหนถูก**

เพลงที่ยังต้องตรวจซ้ำ (⛔ **ยังไม่ต้องแก้**):
`#3` `#13` `#16` `#32` `#59` `#75` `#79` `#102` `#110` `#111` `#115` `#733` `#737` `#759` `#771`
และเพลงที่ยังไม่มีเลข 2 เพลง

**เหตุที่ยังไม่ส่ง:** ห้องที่ตัดข้ามบรรทัด และห้องยกที่ประกาศไว้แล้ว ทำให้นับพลาดได้ทั้งสองทาง
เครื่องมือคนละตัวจัดการสองเรื่องนี้ต่างกัน จึงต้องรวมให้เป็นตัวเดียวก่อนแล้วไล่ตรวจใหม่

**ที่รู้แน่แล้วจากรอบก่อน** (แต่ยังไม่ได้ทำเป็นใบแจ้งละเอียด): เพลง `#13` `#16` `#115` `#120` `#733` `#771`
มีจุดร่วมกันคือ **ห้องแรกของท่อนใหม่สั้น 1 จังหวะ** = น่าจะเป็นห้องยกที่ไม่ได้ประกาศ **แบบเดียวกับ 3.1 ทั้งหมด**
ถ้า 3.1 แก้แล้วได้ผลดี ที่เหลือน่าจะแก้แบบเดียวกัน
