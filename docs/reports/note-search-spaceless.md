# note-search-spaceless — ค้นตามตัวโน้ต (ตรวจถูกต้อง + พิมพ์ไม่เว้นวรรค)

**Branch:** `note-search-spaceless` · **ไฟล์ที่แตะ:** `src/lib/songSearch.js` + `src/lib/songSearch.test.js` เท่านั้น (logic ล้วน · ไม่แตะ notation.js/NoteRow/UI/SongList)

## สรุป (F60+)
ค้นตามตัวโน้ตทำงานถูกต้องอยู่แล้ว (v1 + v2) — ยืนยันด้วยเทสต์ครอบ v1/v2 + เคสขอบ (ออกเทฟ `.`/`'`, โน้ตค้าง `-`, เขบ็ต `_`, ห้อง `|`, ตัวหยุด `0`). **ของใหม่:** พิมพ์ตัวโน้ต **ไม่เว้นวรรค** ก็เจอ — `5561` = `55 61` = `5 5 6 1` เจอเพลงเดียวกัน โดยไม่กระทบค้นเนื้อร้อง / ชื่อ / เลขเพลง.

## A. ตรวจความถูกต้อง (เดิมทำงานอยู่แล้ว)
- `notesText(content)` อ่านทำนองทั้ง v1 (`content.lines`) และ v2 (`content.stanzas[].lines[]`) แล้วตัด decoration (จุดออกเทฟ/ค้าง/เขบ็ต/คอร์ด) เหลือเลขขั้นล้วน — พิสูจน์ด้วยเทสต์ `notesText — v1 and v2` และ `notesCompact — bare scale-degree reduction` (`.5. 5_ 6 | 3 - 1' 0 7` → `5563107`).
- `5 5 6 1` เจอเพลง v2 · `1 2 3 5` เจอ v1 — เทสต์ `search by notes (B058)` ยังเขียว.
- **ไม่พบบั๊กในเส้นทางค้นโน้ตเดิม** — จึงไม่มีการแก้ correctness เพิ่ม.

## B. ⭐ พิมพ์ไม่เว้นวรรคก็เจอ
**ปัญหาเดิม:** `notesText` เก็บโน้ตเป็นเลขคั่นช่องว่าง (`5 5 6 1`) → `5561` เดิมเจอได้ **โดยบังเอิญ** ผ่าน fuzzy (query ยาว ≥4 ตัว บวก compact ทั้ง haystack) แต่ (1) **โน้ตสั้นไม่เจอ** (`561`, `13` → ว่าง) และ (2) ไม่แม่น (fuzzy รับ typo · เทียบทั้ง haystack ปนเนื้อ/ชื่อ).

**วิธีแก้ — เพิ่มเส้นทางค้นโน้ตเฉพาะ (แม่น · ADD-only):**
- `notesCompact(content)` = `notesText` ตัดทุกอย่างที่ไม่ใช่ `0-7` → `5 5 6 1 3` เป็น `55613`.
- `isNoteQuery(q)` = ตัดสินว่า query นี้ = โน้ตหรือเปล่า: ต้องเป็น glyph โน้ตล้วน (`0-7`, จุดออกเทฟ, `- _ | ~ # b ( ) ♮ { }`) และมีขั้นเสียง `1-7` อย่างน้อยหนึ่ง. **กันชนเลขเพลง:** เลขล้วนไม่มีเว้นวรรค/decoration นับเป็นโน้ตต่อเมื่อยาว ≥ 4 ตัว (เลขเพลง ≤ 3 หลัก) → `42`/`100`/`117` = ค้นเลขเพลงเหมือนเดิม · `5561` = โน้ต.
- ใน `scoreSong`: ถ้า `isNoteQuery(q)` → เทียบ `q` (ตัดเหลือ `0-7`) เป็น substring ของ `notesCompact` แบบ **exact (ไม่ fuzzy)** → คืน score 0. รันเฉพาะ query ที่เป็นโน้ต จึง**ไม่แตะ**เส้นทางค้นเนื้อ/ชื่อ/เลข.

**ผลลัพธ์ (พิสูจน์ด้วยเทสต์):**
| พิมพ์ | ได้ |
|---|---|
| `5 5 6 1` · `55 61` · `5561` | เพลงเดียวกัน (v2) ✅ |
| `.5.5.6` (มีจุดออกเทฟในคำค้น) | ตัด decoration → `556` เจอ ✅ |
| `1235` · `12 35` | เพลง v1 ✅ |
| `6666` (ไม่มีในเพลง) | ไม่เจอ (ไม่ false positive) ✅ |
| `12` (เลขเพลง · มีเพลงที่โน้ตขึ้นต้น 1 2) | ได้แค่เพลงเลข 12 — note path ไม่รบกวน ✅ |
| `เยซู` / `มั่นคง` / `พระเจ้าดี` | ค้นเนื้อ/ชื่อเหมือนเดิม ✅ |
