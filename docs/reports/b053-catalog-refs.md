## สรุป (F60+)
การ์ดในหน้ารายการเพลงตอนนี้โชว์ **"แหล่งเพลง" เป็นชื่อเล่มจริง** (map รหัส→ชื่อที่เดียว · เช่น `ล`→เล่มเล็ก) และ **ข้อพระคัมภีร์ (scripture)** ถ้ามี · **ค้นด้วยเลขเล่มเดิม** ได้แล้ว — พิมพ์ `ล.282` / `ล 282` / `ล282` / `เล่มเล็ก 282` เจอเพลงที่มี book_ref นั้น **แบบแม่น** (ไม่ลากเพลงข้างเคียง) · ธีมครบ 8 อยู่แล้ว (derive จากฐาน) · ยืนยันบนเบราว์เซอร์กับ Supabase จริง 121 เพลง.

## 1. map รหัสเล่ม → ชื่อจริง (ที่เดียว)
- ไฟล์ใหม่ `src/lib/bookCodes.js` = SSOT ของชื่อเล่ม (จาก `docs/pm/book-codes.md` · P'Aim ยืนยันครบ 9 เล่ม):
  `ล`=เล่มเล็ก · `ย`=เล่มเยอรมัน · `ยอ`=เยอรมันอนุชน · `ทอ`=ไทยอนุชน 120 · `ม`=เล่มมงคลสมรส · `ส`=เล่มสิงคโปร์ · `สอ`=สิงคโปร์อนุชน · `อ`=อังกฤษ Hymn · `ว`=เล่มไว้อาลัย.
- helper: `bookName(code)` (fallback = รหัสดิบถ้าไม่รู้จัก) · `bookRefLabel({book,no})`→`"เล่มเล็ก 282"` · `bookRefLabels(book_refs[])`→รายการ label เรียงตามเดิม.
- **เปลี่ยนชื่อเล่มแก้ที่เดียว** — การ์ด/ค้นหา/ที่อื่นตามหมด.

## 2. การ์ดโชว์ แหล่งเพลง + scripture
- `SongList.vue`: select เพิ่ม `scripture` (เดิมมี `book_refs` แล้ว) · การ์ดเพิ่ม 2 บรรทัด caption:

  - `📖 <scripture>` (ซ่อนถ้า null)
- สไตล์ = scoped ของ SongList เท่านั้น (น้ำหนัก caption `--fs-xs` เท่า theme-tag · แต่ละอันคนละบรรทัด wrap ได้บนจอมือถือ) — **ไม่แตะ token/สี**.

## 3. ค้นด้วยเลขเล่มเดิม (แม่น)
- `book_refs` เก็บเป็น `[{book,no}]` (code) · เพิ่มใน `songSearch.js`:
  - `bookRefsText(book_refs)` → ต่อเข้า haystack ทุกรูป (`ล.282` `ล 282` `ล282` `เล่มเล็ก 282` เลขล้วน) + scripture → ค้นชื่อเล่ม/เลขแบบหลวมได้ (เช่นพิมพ์ `เล่มเล็ก` เจอทุกเพลงในเล่มเล็ก).
  - `parseBookRefQuery(q)` (ใน `bookCodes.js`) → ถ้า query = **รหัส/ชื่อเล่ม + เลข** (`ล.282`/`ล 282`/`ล282`/`เล่มเล็ก 282`) แปลงเป็น `{book,no}` · ต้องเป็นเล่ม**ที่รู้จักเท่านั้น** (เลขล้วน/เนื้อ/โน้ต `5 5 6 1`/scripture ไม่เข้าเงื่อนไข).
  - ใน `scoreSong`: ถ้าเป็น book-ref query → เทียบ `book_refs` **แบบ exact + ข้าม fuzzy** → คืน 0 เฉพาะเพลงที่มี ref เป๊ะ · กันเพื่อนบ้านคลาด 1 หลัก (`ล.281`) ไม่ให้ปน (fuzzy budget 1 เดิมลากมา 6 เพลง → ตอนนี้ 1 เพลง).
- **cross-ref ข้ามเล่ม:** เพลง 100 มีทั้ง `ล.282` และ `ย.274` → พิมพ์อันไหนก็เจอเพลงเดียวกัน.
