เพลงจริง #1 · เปียโน(ทำนอง+ไล่คอร์ด) + สตริงแพด + เชลโล · สลับกฎเปิด/ปิด แล้วฟังเทียบ

⚠️ อันนี้แค่ทดลองให้เห็นว่ากฎช่วยยังไง — ไม่ release · ตัวที่จะขึ้นจริงคือ เปียโน + กีตาร์ เดี่ยว

เลือกโหมด

❌ กฎปิด (แบบจืด)ทุกเครื่องดังเท่ากัน · แห้งเท่ากัน · เปิดครบทุกท่อน · นิ่ง
✅ กฎเปิด (3 ชั้น)มิติหน้า-หลัง + เพิ่ม/ลดเครื่องตามท่อน + หลบพระเอก
แตะ "เล่น" ครั้งแรกจะโหลดเสียง 3 เครื่อง (~2-4 วิ)

70

กฎ 3 ชั้น (ที่เปิดในโหมด ✅)

  • ชั้นนำ (ทำนอง): เปียโนเล่นทำนองชิ้นเดียว · โน้ตยาว = สเวลล์ (ค่อยดัง)
  • ชั้นเคลื่อนไหว: เปียโนไล่คอร์ด (arp) = ชีพจร · โน้ตทำนองยาว → เติมไล่ (หลบ-เติมพระเอก)
  • ชั้นโอบอุ้ม: สตริง/เชลโล −9dB ใต้ทำนอง + reverb ผลักไป "ด้านหลัง" (เปียโนอยู่หน้า)
  • ตามท่อน: ท่อนต้น (verse) = เปียโน+เชลโลเบา · ท่อนหลัง (chorus) = สตริงแพดเอ่อล้นเข้ามา
