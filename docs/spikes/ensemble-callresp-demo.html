🏛️ เต็มวง Option 1 — เปียโนนำ + ไวโอลินสอดรับ

เพลงจริง #1 · ไม่มี pad สายค้าง (หายออร์แกน) · ไวโอลิน = นักดนตรีคนที่ 2 ที่ เงียบตอนเปียโนเล่น → สวนตอนเปียโนพัก

🎼 กฎ 3 ข้อ: (1) ตัด pad ค้างหมด (2) call-and-response: เปียโนนำ ไวโอลินเงียบ → เปียโนจบวรรค ไวโอลินสีสวนเติมช่องว่าง (3) ท่อนฮุก: ไวโอลินเล่น countermelody สลับจังหวะ หลบย่านเสียงเปียโน

🎻 ชั้นไวโอลิน (สลับเปิด/ปิดฟังทีละกฎ)

✅ สอดรับ (call-response)เปียโนจบวรรค → ไวโอลินสีสวนในช่องว่าง
ปิดสอดรับเปียโนทริโอเปล่า ๆ
✅ Countermelody ท่อนฮุกฮุก → ไวโอลินเล่นไลน์ประสานคู่เปียโน
ปิด countermelodyฮุกไม่มีเส้นไวโอลินประสาน
แตะ "เล่นวง" ครั้งแรกจะโหลดเสียงจริง 3 เครื่อง (~5-8 วิ)
