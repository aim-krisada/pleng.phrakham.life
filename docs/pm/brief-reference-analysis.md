## สิ่งที่ต้องวัด (แมปตรงกับปุ่มของวาทยกรเรา)

เป้าคือทุกบรรทัดต้อง **ชี้ไปที่พารามิเตอร์จริงใน `src/lib/arranger/`** ได้ (`dynamics.js` `patterns.js` `techniques.js` `referee.js` `voicing.js` `embellish.js`)

| วัดอะไร | ทำไม | ไปโผล่ที่ไหนในโค้ดเรา |
|---|---|---|
| **Tempo + rubato curve** (BPM ต่อห้อง · ยืด-หย่อนท้ายวรรค) | คนจริงไม่เล่นตรงเป๊ะ ท้ายวรรคช้าลง | humanize / rubato |
| **Dynamics arc** (ความดังตลอดเพลง · ต่อวรรค) | ค่อยไปดังไปค่อย = อารมณ์ | `dynamics.js` |
| **เชลโลเข้าเมื่อไหร่** (วินาที/ห้องที่เสียงย่านต่ำ-กลางโผล่) | ไม่เข้าตั้งแต่ห้องแรก = เว้นให้เพลงตั้งตัว | โครง section |
| **เปียโนบางลงไหมตอนเชลโลสีทำนอง** (ความหนาแน่นโน้ต/วินาที ก่อน-หลังเชลโลเข้า) | **นี่คือหัวใจของ "เข้ากัน"** — คนจริงหลบให้กัน | `referee.js` (วาทยกร) |
| **การแยกย่านเสียง** (สเปกตรัม: เชลโลอยู่ย่านไหน เปียโนอยู่ย่านไหน) | ไม่ทับกัน = ไม่ตีกัน | `voicing.js` |
| **หางเสียง/reverb + pedal ring** (decay time) | ความ "อิ่ม" | reverb config |
| **ความยาววรรค** (phrase length เป็นห้อง) | เชลโลหายใจตรงไหน | phrase map |

**เครื่องมือ:** `py -3.14` (มี ffmpeg แล้ว · **librosa/demucs ยังไม่ได้ติดตั้ง** → `pip install librosa` ก่อน) · แยกเสียงเชลโลออกจากเปียโน = `demucs` (ถ้าติดตั้งไหว ทำให้วัดแม่นขึ้นมาก · ถ้าหนักไป → รายงานว่าทำไม่ได้ อย่าเดา)
