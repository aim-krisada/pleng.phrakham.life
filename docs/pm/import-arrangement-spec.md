## หลักการหลัก (P'Aim)
- **ทำนองข้อที่ใช้ซ้ำทุกข้อ = dedup เป็น stanza เดียว** · รับที่ใช้ทำนองเดียว = stanza เดียว (หัวใจ v2)
- **เนื้อข้อ 2/3/… ไม่กระโดด** — ร้องไล่ตรง (2.1→2.2→2.3→2.4) · การกระโดด/ย้อน เป็นเรื่อง**การเขียนโน้ตของข้อ 1 ข้อเดียว** (parser จัดให้ข้ออื่น map ตรงลงทำนองเดียวกัน)

## เพลง 100 (hardest — ย้อนกลับ + ซ้ำรับ) · จากที่ P'Aim อธิบาย
- **โน้ตข้อ 1 เขียนแบบกระโดด/ย้อน:** ร้องบรรทัด 1.1 → 1.3 → 1.2 → 1.4 (เพราะโน้ตเล่นมาถึงแล้วต้องย้อนขึ้นไปเล่นอีกท่อน)
- **รับ:** รับ 1.1 → รับ 1.3 → รับ 1.2 → รับ 1.4 (จุด 4) แบบเดียวกัน
- **ข้อ 2/3:** เนื้อ**ไม่กระโดด** — ร้อง 2.1 → 2.2 → 2.3 → 2.4 ตรงๆ แล้วมารับ (รับเหมือนเดิม) · ข้อ 3 = 3.1→3.2→3.3→3.4 แล้วรับ
- **สรุป:** การกระโดด/ย้อน อยู่ที่**การเขียนโน้ตข้อ 1 เท่านั้น** · ข้อ 2/3 map เนื้อไล่ตรงลงทำนองเดียวกัน · parser ต้องเข้าใจ repeat/ย้อน ของข้อ 1 แล้วกาง arrangement ให้ข้ออื่นร้องตรง
- **P'Aim: "ถ้าไม่เข้าใจก็บอก"** → DA ศึกษา source เพลง 100 + transcript นี้ · ถ้ายังไม่ชัวร์ให้ถาม P'Aim ก่อนทำ (อย่าเดา)
