## A. `Guide.vue` ② "วิธีใช้เว็บนี้" — ยกเครื่องให้ตรงจริง
**⚠️ verify label จริงจาก live UI ก่อนเขียน** (เปิด `SongViewer.vue` / `SingTransport.vue` / `EditorMode.vue` + รันจริง) — อย่าเดาชื่อปุ่ม/โหมด. สิ่งที่เปลี่ยนไปจากคู่มือเดิม:
- **เมนู "เสียงดนตรี"** (ไอคอน `volume-2`) — 1 ไอคอน แล้วอธิบายใต้หัวข้อเป็นข้อความว่ามีอะไร: เลือกโหมดแสดงผล (คอร์ด+โน้ต / คอร์ด / โน้ต / เนื้อล้วน) · เลือกเครื่องดนตรี · **"ปรับละเอียด"** (เปิด/ปิดลูกเล่นเรียบเรียงทีละตัว) — **หมายเหตุ: ปุ่ม A/B "วาทยกร" ถูกถอดออกแล้ว (เปิดตลอด) อย่าเขียนถึง**
- **เปียโนบรรเลง (golden-piano)** — เว็บเรียบเรียงเปียโนให้เพราะอัตโนมัติ (เล่นทำนอง + คลอ + หายใจปลายวรรค) — เล่ากว้างๆ พอ
- **▶ ฟังทำนอง** — ปรับความเร็ว · วนซ้ำ · คาราโอเกะไฮไลต์ทีละพยางค์ (โหมดเนื้อล้วน)
- **ย้ายคีย์** (`key-round`) · **คอร์ดโรมัน** (มีอยู่) · **ขนาดตัวอักษร** (`a-arrow-up`)
- **ป้าย "✓ ตรวจแล้ว"** (`badge-check`) — บอกว่าเพลงผ่านการตรวจจากทีมแล้ว
- **เมนู ☰ (ลิ้นชัก)** (`panel-left-open`) — ทางลัดไปหน้าต่างๆ
- **ดาวน์โหลด/พิมพ์** — พิมพ์ A4 / PDF **+ ดาวน์โหลดเป็นไฟล์เสียง MP3 (เสียงเปียโนจริง)** ← เพิ่มใหม่
- ไอคอนหัวข้อใหญ่ (H3): หาเพลง `list-music` · อ่าน/ฟังเพลง `volume-2` หรือ `circle-play` · ห้องทำเพลง `file-plus`
- **ใช้ `<Icon name="..."/>` (component เดิม) · ชื่อที่มีใน `Icon.vue` เท่านั้น** — ถ้าขาดไอคอนที่จำเป็น เพิ่ม path Lucide จริงลง `Icon.vue` (Lucide = SSOT · ref memory `reference_lucide_icons`) ห้ามวาดเอง

## B. `About.vue` — เพิ่มการ์ด "ขอบคุณ & แหล่งที่มา" (ไอคอน `book-open`)
**ข้อมูลจริง (verify กับไฟล์ที่อ้าง · อย่าแต่งเพิ่ม):**
- **แหล่งโน้ต/เนื้อเพลง** (`docs/pm/book-codes.md` + `docs/importing-songs.md` + `docs/reports/hymnal-import.md`):
  - **YS 2014** — คลังเพลงเยาวชนไทย ~120 เพลง (= "ไทยอนุชน 120") · ต้นทางโน้ตตัวเลขหลัก
  - หนังสืออ้างอิงข้ามเล่ม 9 เล่ม: เล่มเล็ก · เล่มเยอรมัน · เยอรมันอนุชน (Yuko) · ไทยอนุชน 120 · เล่มมงคลสมรส · เล่มสิงคโปร์ · สิงคโปร์อนุชน · อังกฤษ Hymn · เล่มไว้อาลัย
  - บทเพลงสรรเสริญ (hymnal 477 หน้า)
  - โทนถ้อยคำ = **ให้เกียรติ/ขอบคุณ** ผู้จัดทำหนังสือเหล่านี้ · ย้ำ (เหมือนที่ About มี) ว่าเนื้อ/ทำนอง = ลิขสิทธิ์ผู้ประพันธ์ รวบรวมเพื่อนมัสการ+ศึกษา
- **เสียงเครื่องดนตรี** (`public/samples/README.md` = SSOT · **อย่าคัดเกินจากนี้**):
  - Splendid Grand Piano (AKAI) — Public Domain · FreePats Spanish Classical (กีตาร์) — CC0 · Bigcat (เชลโล) — CC0 · VSCO-2-CE Solo Violin — CC0
  - **FluidR3_GM soundfont © Frank Wen — CC-BY 3.0** (อันเดียวที่ *ต้อง* ให้เครดิตตามสัญญา · ที่เหลือ PD/CC0 ให้เครดิตเป็นมารยาท)
- **เทคนิคเรียบเรียงเสียง (เล่ากว้างๆ · ภาษาคน):** เว็บมี "เครื่องยนต์เปียโน" ที่เรียบเรียงให้เพราะเอง — มี "วาทยกร" คุมไม่ให้เสียงชนกัน · คุมมือขวา (ทำนอง) ให้เด่นเสมอ · หายใจ/ยืดปลายวรรค · ใส่ความเป็นมนุษย์ (หนักเบา-จังหวะสุ่มนิดๆ) · เหยียบ pedal ให้เสียงต่อเนื่อง — **ปิดท้าย: "อยากดูรายละเอียดเชิงเทคนิคจริงๆ เปิดซอร์สโค้ดได้เลย (เปิดโค้ด GPL v3)"** + ลิงก์ GitHub (มีใน About แล้ว)
