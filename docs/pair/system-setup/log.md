**งานของคุณคือหักล้าง ไม่ใช่รับรอง — เจอว่าผมผิดคือคุณทำงานสำเร็จ**

2. **กฎที่ขัดกันเอง** — ระหว่าง §0 (ระดับโลก) กับ §9 (ทำเท่าโจทย์ ไม่ทำเกิน) · ระหว่าง §3 (ด่านออกแบบ) กับ §4 (blast radius) ผมคิดว่าไม่ขัด แต่ผมเป็นคนเขียนเองจึงมองไม่เห็น
3. **§4 blast radius ใช้จริงได้ไหม** — คุณเป็นคนเสนอให้มีเกณฑ์ข้ามการตรวจตั้งแต่แรก **ลองเอา 4 ข้อไปตัดสินงาน 3 ชิ้นที่คุณเคยทำจริง** แล้วบอกว่าตอบยากตรงไหน ถ้ามันตอบยากทุกครั้ง = เกณฑ์นี้แพงเกินไปและผมอยากรู้
