# DS-C03 — ส่งขออนุมัติทางอีเมล

**คู่กับ:** `us/wt-c-json/US-C03-email-submit.md`

## ไฟล์ที่แตะ
- `src/lib/jsonIO.js` · `src/components/DownloadTool.vue` (หรือปุ่มในเมนูจัดการ)

## จุดเสี่ยงชนกับ worktree อื่น
- ไม่มี (ไฟล์ของ WT-C) — อีเมลทีมเก็บเป็น config/const

## design
- ปุ่ม "ส่งขออนุมัติ" → (1) `exportSong` ให้ผู้ใช้ดาวน์โหลด (2) เปิด `mailto:` ถึงอีเมลทีม พร้อม `subject`/`body` ตั้งต้น + ข้อความ "กรุณาแนบไฟล์ JSON ที่เพิ่งดาวน์โหลด"
- **ข้อจำกัด:** `mailto` แนบไฟล์อัตโนมัติไม่ได้ → ต้องให้ผู้ใช้แนบเอง (สื่อสารให้ชัด)

## test
