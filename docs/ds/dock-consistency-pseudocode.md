# Pseudocode — dock โครงเดียว 3 โหมด (นับปุ่มครบ + ตำแหน่ง + default vs ขยาย)

> เขียนก่อน build — ตรวจพื้นฐานว่ามีกี่ปุ่ม อะไรบ้าง อยู่ตรงไหน · แยกชัด "default โชว์อะไร" กับ "ขยายแล้วมีกลุ่มอะไร"
> ป้ายตำแหน่ง (placement) ของทุกปุ่ม:
> - `FIXED` = อยู่คงที่ทุกโหมด (grip ซ้ายสุด · ⚙ ขวาสุด · ⋯ ขวาก่อน ⚙)
> - `CORE` = ปุ่มหลักของโหมด อยู่บนแถบเสมอ (ถอดไม่ได้)
> - `PERMANENT` = อยู่บนแถบเสมอ ปักถอนไม่ได้ (เช่น Aa)
> - `BAR*` = ปักได้ · default = ปักอยู่บนแถบ
> - `PANEL` = ปักได้ · default = อยู่ในแผงเฟืองอย่างเดียว
> - `ROW` = แถวพิเศษเต็มกว้าง (timeline / คีย์ jianpu)

---

## 0 · ค่าคงที่โครง (ทุกโหมด เหมือนกันหมด)
```
GRID_UNIT      = 1 slot ≈ 44px (ปุ่มปกติ) · ปุ่มกว้าง/ป้าย = 2 slot · timeline = ไอคอนยาว
ROW_CAP        = { mobile: 6 slots, desktop: 12 slots }   // ต่อบรรทัด
ANCHOR_LEFT    = grip        // slot ซ้ายสุดของบรรทัดล่าง เสมอ
ANCHOR_RIGHT   = [⋯?] ⚙      // ⚙ ขวาสุด เสมอ · ⋯ อยู่ซ้ายติด ⚙ เฉพาะตอนล้น
OVERFLOW       = ปุ่มเกิน cap → พับเข้า ⋯ (popover)
POPUP_RULE     = ทุก popover/แผง เปิดเหนือแถบ + **CLAMP ไม่ให้ล้นขอบจอ (+8px)**  // ข้อกำหนดบังคับ
ONE_OPEN       = เปิดได้ทีละ 1 (แผงเฟือง | เลือกท่อน | เมนู | ⋯)  // เปิดใหม่ปิดเก่า
```

## 3 · การแสดงผล — ขยายจาก default (กลุ่มที่โผล่)

```
กด ⚙ (ทุกโหมด) → แผงเฟือง "บ้าน" เปิดเหนือแถบ [CLAMP]:
    ┌ ตั้งค่า — ปรับได้ที่นี่เลย · 📌 = ปักขึ้นแถบ ┐
    │ 🎸 คอร์ด      [กีตาร์ ▾]           📌     │   // PIN แต่ละตัว:
    │ ⏱ ความเร็ว   ◀ 90 ▶      ▲ ▼      📌•    │   //   ค่าปรับ inline
    │ 🔑 คีย์       ◀ C ▶       ▲ ▼      📌•    │   //   ▲▼ = ลำดับบนแถบ (ถ้าปัก)
    │ ▤ แสดงผล     [คอร์ด+โน้ต ▾] ▲ ▼   📌•    │   //   📌• = ปักแล้ว (โผล่บนแถบด้วย)
    │ ◑ ความโปร่ง  [——slider——]         📌     │
    │ ⭳ ดาวน์โหลด  [บันทึก]              📌     │
    │ 🖨 พิมพ์      [เปิด]                📌     │
    └──────────────────────────────────────────┘
    // ปัก 📌 ตัวใหม่ → เพิ่มบนแถบ → ถ้าเกิน cap ก็ขึ้นบรรทัด/เข้า ⋯
    // ถอน 📌 → กลับเข้าแผงอย่างเดียว

กด เลือกท่อน (ฝึกร้อง) → แผงรายการท่อน [CLAMP]:
    [☑ ทั้งหมด] [☐ ไม่เลือก]   (ไม่เลือก = เล่นทั้งเพลง)
    ☑ ท่อน 1   ☑ ท่อน 2   ☑ ฮุก   ☐ ท่อน 3   ☐ จบ

กด คอร์ด/คีย์/ความเร็ว/แสดงผล (บนแถบ) → dropdown เมนู [CLAMP, ยึดใต้ปุ่มนั้น]
กด Aa (บนแถบ) → popover สไลเดอร์ขนาดอักษร [CLAMP]   // ← ตัวที่เคยล้นขอบ ตอนนี้ clamp แล้ว
กด ⋯ (ถ้ามี) → popover ปุ่มที่ล้น [CLAMP]
กด grip → ยุบทั้งแถบเป็นปุ่มลอยกลม (FAB) [CLAMP] · แตะ FAB = กางคืน · ลาก = ย้าย
```

## 4 · อัลกอริทึมเรนเดอร์ (โครงเดียว)
```
render(mode, viewport):
  if collapsed: return FAB[CLAMP]

  rows = []
  # แถวพิเศษเต็มกว้าง (บนสุด)
  if mode == edit:  rows += KEYS_ROW(jianpu)
  if mode == sing:  rows += TIMELINE_ROW( slider[4] + select[2] )

  cap = ROW_CAP[viewport]
  pinned = userPins(mode)                       # default ตาม §1

  # แถว pins (มือถือ = แถวแยกเหนือ bottom · เดสก์ท็อป = ไปรวม bottom)
  if viewport == mobile and pinned:
      rows += PINS_ROW(pinned + [Aa], wrap by cap)

  # แถวล่าง = สมอ (มีเสมอ)
  bottom = [ grip@LEFT ]
  bottom += MAIN(mode)                          # sing:transport · edit:tools · print:[พิมพ์]
  if viewport == desktop: bottom += pinned + [Aa if mode has Aa]
  (overflow, bottom) = FOLD_TAIL(bottom, cap)   # เกิน cap → overflow
  bottom += [ ⋯@RIGHT if overflow ] + [ ⚙@RIGHT ]
  rows += bottom

  # popover (เปิดทีละ 1) — เหนือแถบ + CLAMP เข้าจอเสมอ
  if openPanel == gear:   POPUP = GEAR_PANEL(pinnableList(mode))
  if openPanel == select: POPUP = SELECT_PANEL(sections)
  if openMenu:            POPUP = DROPDOWN(anchored to its bar button)
  CLAMP(POPUP into viewport, margin 8px)        # ← ข้อกำหนดบังคับ ห้ามล้นขอบ

  return rows + POPUP
```

## 5 · Invariants (จริงทุกครั้ง)
```
I1  grip = slot ซ้ายสุดของบรรทัดล่าง เสมอ ทุกโหมด ทุกจอ
I2  ⚙  = slot ขวาสุดของบรรทัดล่าง เสมอ · ⋯ ติดซ้าย ⚙ เฉพาะตอนล้น
I3  ปุ่ม/แถวเพิ่ม → ขึ้นบรรทัดเหนือ · grip/⚙ ไม่ขยับ
I4  แผง ⚙ = บ้านครบทุก PIN ของโหมด · 📌 = สลับขึ้น/ลงแถบ · ปักแล้วยังอยู่ในแผง
I5  ทุก popover/แผง เปิดเหนือแถบ + CLAMP ไม่ล้นขอบ (+8px) — ไม่มีข้อยกเว้น
I6  เปิดได้ทีละ 1 · Esc/แตะนอก = ปิด
I7  Aa = อยู่บนแถบเสมอ ปักถอนไม่ได้ (คนอ่านต้องมี)
I8  tap target ≥ 44px · aria-label ทุกปุ่ม
```
