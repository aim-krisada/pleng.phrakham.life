# US-D05 — ตัวแก้ = แผ่นเดียว (โน้ต+เนื้อ เป็นกล่องพิมพ์ ripple ตรงคอลัมน์)

**Worktree:** WT-D · **Branch:** `wt-d-editor-library` · **ไฟล์:** `src/components/EditorMode.vue`
**สถานะ:** dev เสร็จ (รอ tester)
**โยง mission:** คนทำเพลงแก้โน้ต+คำได้ในที่เดียว อ่านง่าย ไม่ตกจอ → ทำเพลงเข้าคลังเร็ว/มั่นใจ
**ที่มา:** พี่เอม 2026-07-08

## เรื่องราว / การตัดสินใจ
เดิมตัวอย่าง "แผ่นเพลง" เป็นก้อนแยก (บนสุด แล้วลองย้ายมาเหนือแต่ละบรรทัด) → กลายเป็น
**2 ชั้นซ้อน** ข้อมูลเดียวกันแต่โชว์ 2 รอบ ทำให้สูงเป็น 2 เท่า ตกจอ ต้องเลื่อน และไม่ตรง
คอลัมน์กับกล่องแก้.

**ข้อสรุปของพี่เอม:** ไม่ต้องมีแผ่น preview แยก · ไม่แตะโน้ตเพื่อแก้ · ให้ **ตัวแก้เอง**
เป็นแผ่นเดียวที่ทั้งดูและแก้: โน้ตเป็นกล่องพิมพ์ (ripple) และเนื้อร้องเป็นกล่องพิมพ์ราย
พยางค์ใต้โน้ตแต่ละตัว (ripple) เรียงตรงคอลัมน์กัน คอร์ดอยู่บน.

## User Story
**ในฐานะ** คนทำเพลงของคริสตจักร
**ฉันต้องการ** แก้โน้ตและคำร้องในกล่องที่วางตรงคอลัมน์กัน ในที่เดียว (ไม่มีแผ่นซ้ำ)
**เพื่อ** อ่าน+แก้ได้ตรงจุด ไม่ต้องเลื่อนจอ ไม่มีของซ้ำให้สับสน

## Acceptance Criteria
- [x] เอาแผ่น preview ที่ซ้ำออก (ทั้งก้อนบนสุดและรายบรรทัด) → ไม่ตกจอ
- [x] โน้ต = กล่องพิมพ์ ripple (พิมพ์แล้วขึ้นช่องถัดไป · ลบแล้วยุบ) — *ของเดิม คงไว้*
- [x] เนื้อร้อง = กล่องพิมพ์รายพยางค์ใต้โน้ตแต่ละตัว ripple (แยกพยางค์/ดัน/ดึง) — *ของเดิม คงไว้*
- [x] โน้ตกับคำ ตรงคอลัมน์เดียวกัน คอร์ดอยู่บน = อ่านเป็นแผ่นได้ในตัว
- [x] ดูแผ่นเต็มทั้งเพลง = ปุ่มโหมด "แผ่น" (🎼)
