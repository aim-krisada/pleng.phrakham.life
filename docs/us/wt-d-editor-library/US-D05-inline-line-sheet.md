# US-D05 — ตัวอย่างแผ่นเพลงแทรกรายบรรทัด (แก้ตรงไหน เห็นแผ่นตรงนั้น)

**Worktree:** WT-D · **Branch:** `wt-d-editor-library` · **ไฟล์:** `src/components/EditorMode.vue`
**สถานะ:** dev เสร็จ (รอ tester)
**โยง mission:** คนทำเพลงเห็นผลลัพธ์จริงระหว่างแก้ → ทำเพลงเข้าคลังได้มั่นใจ/เร็วขึ้น
**ที่มา:** พี่เอม 2026-07-08 — เดิมตัวอย่างแผ่นเป็นก้อนเดียวบนสุด แยกจากจุดที่กำลังแก้

## User Story
**ในฐานะ** คนทำเพลงของคริสตจักร
**ฉันต้องการ** เห็นตัวอย่าง "แผ่นเพลง" ของบรรทัดที่กำลังแก้ อยู่เหนือช่องแก้ของบรรทัดนั้นเลย
**เพื่อ** เทียบโน้ต/คำที่พิมพ์กับหน้าตาแผ่นจริงได้ทันทีตรงจุด ไม่ต้องเลื่อนขึ้นไปดูก้อนรวมด้านบน

## Acceptance Criteria
- [x] เอาก้อนตัวอย่างแผ่นรวมด้านบนสุดออก
- [x] แต่ละบรรทัด (ท่อนที่แก้) มีตัวอย่างแผ่นของ *บรรทัดนั้น* แสดงเหนือช่องโน้ต — อัปเดตสดขณะพิมพ์
- [x] คำร้องในตัวอย่างมาจากข้อ (ข้อ) ที่เลือก ตัดเฉพาะพยางค์ของบรรทัดนั้น
- [x] ดูแผ่นเต็มทั้งเพลงได้จากปุ่มโหมด "แผ่น" (🎼) เหมือนเดิม
