# US-D04 — จัดเก็บเป็นระบบ: หมวด + เลขในหมวด

**Worktree:** WT-D · **Branch:** `wt-d-editor-library` · **คู่ DS:** `ds/wt-d-editor-library/DS-D04-category-number.md`
**สถานะ:** spec (รอ dev) · **ใหม่ (data-model change)**
**โยง mission:** คลังเป็นระบบ ค้นเจอ เลขไม่ซ้ำในหมวด

## User Story
**ในฐานะ** เจ้าของระบบ / ผู้อนุมัติ
**ฉันต้องการ** ให้เพลงหนึ่งอยู่ได้หลายหมวด แต่ละหมวดมีเลขของตัวเอง (ไม่ซ้ำในหมวด)
**เพื่อ** ให้คลังเป็นระบบ อ้างเลขได้เหมือนหนังสือเพลง (เช่น #4 ยุวชน · #12 อนุชน)

## Acceptance Criteria
- [ ] Properties ตั้ง หมวด+เลข ได้หลายหมวด (`song.categories = [{name, number}]`)
- [ ] รายการเพลงเรียงตามเลขในหมวดที่เลือก
- [ ] เลขไม่ซ้ำในหมวดเดียวกัน (เตือนถ้าซ้ำ)

## นอกขอบเขต
- ย้าย/รวมหมวดจำนวนมากทีเดียว (ทำภายหลังถ้าต้องการ)
