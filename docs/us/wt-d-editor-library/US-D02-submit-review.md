# US-D02 — ส่งเพลงให้ตรวจ

**Worktree:** WT-D · **Branch:** `wt-d-editor-library` · **คู่ DS:** `ds/wt-d-editor-library/DS-D02-submit-review.md`
**สถานะ:** spec (รอ dev)
**โยง mission:** เพลงเข้าคลังอย่างเป็นระบบ (ผ่านการตรวจ)

## User Story
**ในฐานะ** คนทำเพลง
**ฉันต้องการ** ส่งเพลงที่ทำเสร็จให้ผู้อนุมัติตรวจ
**เพื่อ** ให้เพลงได้เข้าคลังกลาง

## Acceptance Criteria
- [ ] ปุ่ม "ส่งตรวจ" เปลี่ยนสถานะเป็น pending *(มีแล้ว)* — ผ่าน `save('pending')`
- [ ] ผู้อนุมัติเห็นรายการที่รอตรวจ

## นอกขอบเขต
- การแจ้งเตือนอีเมลถึงผู้อนุมัติ (ทำภายหลังถ้าต้องการ)
