# US-D03 — อนุมัติ / ตีกลับ / ลบ / ย้อน

**Worktree:** WT-D · **Branch:** `wt-d-editor-library` · **คู่ DS:** `ds/wt-d-editor-library/DS-D03-approve-manage.md`
**สถานะ:** spec (รอ dev)
**โยง mission:** คุมคุณภาพคลังกลาง (เฉพาะผู้อนุมัติ)

## User Story
**ในฐานะ** ผู้อนุมัติ (approver)
**ฉันต้องการ** ดูเพลงที่รอตรวจ แล้วอนุมัติเข้าคลัง หรือตีกลับให้แก้ / ลบ / ย้อนเวอร์ชัน
**เพื่อ** คุมคุณภาพคลังกลางให้เชื่อถือได้

## Acceptance Criteria
- [ ] อนุมัติ → เพลงเข้าคลัง (approved) แสดงในรายการสาธารณะ *(มีแล้ว)*
- [ ] ตีกลับ (ให้แก้) / ลบ / ย้อนประวัติ *(มีแล้ว)*
- [ ] เฉพาะ approver เห็นปุ่มเหล่านี้ (`store.canApprove`)

## นอกขอบเขต
- ระบบ comment/review แบบละเอียด (ทำภายหลังถ้าจำเป็น)
