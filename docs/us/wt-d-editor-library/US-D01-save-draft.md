# US-D01 — ทำ/แก้เพลง บันทึกร่างในระบบ

**Worktree:** WT-D · **Branch:** `wt-d-editor-library` · **คู่ DS:** `ds/wt-d-editor-library/DS-D01-save-draft.md`
**สถานะ:** spec (รอ dev)
**โยง mission:** คนทำเพลงของ คจ. เพิ่มเพลงเข้าคลังอย่างเป็นระบบ

## User Story
**ในฐานะ** คนทำเพลงของคริสตจักร (ล็อกอิน)
**ฉันต้องการ** ทำเพลงใหม่หรือแก้เพลงเดิม แล้วบันทึกเป็นร่างในระบบ
**เพื่อ** เก็บงานไว้ทำต่อได้ทุกที่ ไม่หาย

## Acceptance Criteria
- [ ] โหมดแก้ (`EditorMode`) ทำเพลง/แก้ได้ *(มีแล้ว)*
- [ ] บันทึกร่าง (draft) ลงระบบ *(มีแล้ว)* — ผ่าน `save('draft')` ตาม contract
- [ ] เปิดร่างเดิมมาทำต่อได้

## นอกขอบเขต
- การรื้อ UI ภายใน editor (คงของเดิม เว้นแต่จุด เอื้อน/slur ที่แยกเป็น story ภายหลัง)
