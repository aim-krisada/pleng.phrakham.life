# US-C03 — ส่งเพลงขออนุมัติทางอีเมล

**Worktree:** WT-C · **Branch:** `wt-c-json` · **คู่ DS:** `ds/wt-c-json/DS-C03-email-submit.md`
**สถานะ:** spec (รอ dev)
**โยง mission:** คนทำเพลงภายนอกเสนอเพลงเข้าคลังได้ โดยไม่เก็บอะไรในระบบ

## User Story
**ในฐานะ** คนทำเพลงภายนอก (ไม่มีสิทธิ์เข้าคลัง)
**ฉันต้องการ** ส่งเพลงที่ทำ (เป็นไฟล์ JSON) ไปขออนุมัติทางอีเมล
**เพื่อ** เสนอเพลงเข้าคลังได้ โดยไม่ต้องมีบัญชีในระบบ

## Acceptance Criteria
- [ ] มีปุ่ม/ขั้นตอน "ส่งขออนุมัติ": (1) ดาวน์โหลด JSON ให้ก่อน (2) เปิดอีเมล (mailto) ถึงทีม พร้อมหัวข้อ/ข้อความตั้งต้น
- [ ] มีข้อความบอกผู้ใช้ให้ **แนบไฟล์ JSON ที่เพิ่งดาวน์โหลด** (mailto แนบไฟล์เองอัตโนมัติไม่ได้)
- [ ] ระบบไม่เก็บอะไร (ส่งนอกระบบล้วน)

## นอกขอบเขต
- ระบบรับ-คิวอนุมัติในเว็บ (จงใจไม่ทำ — ใช้อีเมลตามที่เจ้าของกำหนด)
