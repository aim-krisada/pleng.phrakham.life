# US-C01 — ดาวน์โหลดเพลงเป็น JSON

**Worktree:** WT-C · **Branch:** `wt-c-json` · **คู่ DS:** `ds/wt-c-json/DS-C01-download-json.md`
**สถานะ:** spec (รอ dev)
**โยง mission:** คนทำเพลงภายนอกใช้เครื่องมือได้ผ่าน JSON ของตัวเอง ไม่ปนคลัง

## User Story
**ในฐานะ** คนใช้เว็บที่ไม่ได้ล็อกอิน (คนทั่วไป / คนทำเพลงภายนอก)
**ฉันต้องการ** ดาวน์โหลดเพลงที่กำลังทำเป็นไฟล์ JSON เก็บไว้เอง
**เพื่อ** เก็บงานของตัวเองไว้ทำต่อ โดยไม่ต้องพึ่งคลังกลาง

## Acceptance Criteria
- [ ] มีปุ่มดาวน์โหลด JSON (ทุก tier) *(มีแล้วบางส่วน)*
- [ ] ไฟล์ที่ได้เปิดกลับมาได้ (คู่กับ US-C02) — round-trip ครบ
- [ ] ชื่อไฟล์สื่อความ (อิงชื่อเพลง)

## นอกขอบเขต
- การเก็บลงคลังกลาง (เฉพาะทีม — WT-D)
