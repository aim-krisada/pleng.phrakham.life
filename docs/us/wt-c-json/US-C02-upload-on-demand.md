# US-C02 — อัปโหลด JSON มาเปิด (on-demand)

**Worktree:** WT-C · **Branch:** `wt-c-json` · **คู่ DS:** `ds/wt-c-json/DS-C02-upload-on-demand.md`
**สถานะ:** spec (รอ dev)
**โยง mission:** เว็บเป็นเครื่องมือฟรี · unprivileged เก็บงานเป็น JSON ของตัวเอง

## User Story
**ในฐานะ** คนใช้เว็บที่ไม่ได้ล็อกอิน (รวมคนที่อยากเอาเพลงตัวเองมาฝึกร้อง)
**ฉันต้องการ** อัปโหลดไฟล์ JSON ที่มี เข้ามาเปิดดู / ฟัง / แก้ แบบชั่วคราว
**เพื่อ** ใช้เพลงของตัวเองบนเครื่องมือของเว็บ โดยระบบไม่เก็บอะไรไว้

## Acceptance Criteria
- [ ] อัปโหลด JSON → เปิดเป็นเพลงใน Studio (ดู/ฝึก/แก้) *(มีแล้ว `manageUpload`)*
- [ ] ระบบ **ไม่บันทึกลงฐานข้อมูล** (on-demand ล้วน)
- [ ] ไฟล์เสีย/ผิดรูป → แจ้งเตือน ไม่พัง (ดู US-C04)

## นอกขอบเขต
- การ sync/บันทึกข้ามเครื่อง (นั่นคือ tier ที่ล็อกอิน)
