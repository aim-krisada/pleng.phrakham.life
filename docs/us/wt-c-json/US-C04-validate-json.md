# US-C04 — ตรวจไฟล์ JSON ก่อนเปิด

**Worktree:** WT-C · **Branch:** `wt-c-json` · **คู่ DS:** `ds/wt-c-json/DS-C04-validate-json.md`
**สถานะ:** spec (รอ dev)
**โยง mission:** เข้าถึงง่าย ไม่พัง · SSOT = jianpu มาตรฐาน (ไฟล์ต้องถูกโครง)

## User Story
**ในฐานะ** คนที่อัปโหลด JSON
**ฉันต้องการ** ให้ระบบตรวจไฟล์ก่อนเปิด ถ้าผิดรูปบอกให้รู้
**เพื่อ** ไม่ให้เว็บค้าง/พังเพราะไฟล์เสีย

## Acceptance Criteria
- [ ] ตรวจว่าเป็น JSON ถูกโครงเพลง (v1/v2) ก่อนเปิด
- [ ] ไฟล์ผิด → ข้อความบอกสาเหตุแบบเข้าใจง่าย ไม่ crash
- [ ] รองรับทั้ง v1 และ v2 (v1 แปลงเป็น v2 อัตโนมัติ)

## นอกขอบเขต
- แก้ไฟล์ที่เสียให้อัตโนมัติ (แค่บอกว่าเสีย)
