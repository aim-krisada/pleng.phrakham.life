# US-A02 — ปรับคีย์ / ความเร็ว / วนซ้ำ

**Worktree:** WT-A · **Branch:** `wt-a-sing` · **คู่ DS:** `ds/wt-a-sing/DS-A02-key-tempo-loop.md`
**สถานะ:** spec (รอ dev)
**โยง mission:** ฝึกร้องได้เอง ตามระดับเสียงของตัวเอง

## User Story
**ในฐานะ** คนฝึกร้อง
**ฉันต้องการ** ปรับคีย์ให้เข้ากับเสียงตัวเอง ปรับความเร็ว และวนซ้ำท่อนที่ยังไม่คล่อง
**เพื่อ** ฝึกได้ตามระดับเสียงและความพร้อมของตัวเอง

## Acceptance Criteria
- [ ] ทรานสโพสคีย์ขึ้น/ลง เห็นผลทันที *(มีแล้ว)*
- [ ] ปรับความเร็ว (tempo) *(มีแล้ว)*
- [ ] วนซ้ำช่วงที่เลือก (loop) *(มีแล้ว)*
- [ ] ปรับแล้วไม่กระทบเพลงต้นฉบับ (เป็นการดูชั่วคราว)
