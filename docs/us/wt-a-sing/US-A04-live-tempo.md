# US-A04 — เปลี่ยนความเร็ว (tempo) ตอนกำลังเล่น

**Worktree:** WT-A · **Branch:** `wt-a-sing` · **Port:** 5302 · **คู่ DS:** `ds/wt-a-sing/DS-A04-live-tempo.md`
**สถานะ:** spec (รอ dev) · **ที่มา:** review WT-0 ข้อ 5 (พี่เอม) · **feature ใหม่ ไม่ใช่ regression**
**โยง mission:** ฝึกร้องได้ตามจังหวะของตัวเอง

## User Story
**ในฐานะ** คนฝึกร้อง
**ฉันต้องการ** ปรับความเร็วระหว่างที่เพลงกำลังเล่น แล้วให้เล่นต่อจากจุดปัจจุบันด้วยความเร็วใหม่
**เพื่อ** เร่ง/ผ่อนจังหวะได้ทันทีระหว่างฝึก ไม่ต้องหยุดแล้วเริ่มใหม่ตั้งแต่ต้น

## Acceptance Criteria
- [ ] เปลี่ยน tempo ตอนเล่น → เล่นต่อจากบรรทัด/โน้ตปัจจุบันที่ bpm ใหม่ (**ไม่เด้งกลับต้นเพลง**)
- [ ] tempo ก่อนเล่น (มีอยู่แล้ว) ยังทำงานปกติ

## นอกขอบเขต
- ทำให้ seamless แบบ detune ของ key (tempo ทำแบบนั้นไม่ได้ — ต้อง re-schedule โน้ตที่เหลือ ยอมรับสะดุดเล็กน้อย)
