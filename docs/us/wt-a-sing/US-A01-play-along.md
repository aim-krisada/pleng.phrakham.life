# US-A01 — เล่นตามแบบคาราโอเกะ

**Worktree:** WT-A · **Branch:** `wt-a-sing` · **คู่ DS:** `ds/wt-a-sing/DS-A01-play-along.md`
**สถานะ:** spec (รอ dev)
**โยง mission:** วัดความสำเร็จ — คนทั่วไปเปิดเพลงแล้วร้อง/ฝึกได้เอง

## User Story
**ในฐานะ** คนที่อยากฝึกร้องเพลงนมัสการ
**ฉันต้องการ** กดเล่นเพลงแล้วเห็นไฮไลต์วิ่งตามโน้ต/คำที่กำลังร้อง
**เพื่อ** ร้องตามได้ทันจังหวะ เหมือนคาราโอเกะ

## Acceptance Criteria
- [ ] กดเล่น → เสียงเล่นตามโน้ต + ไฮไลต์ตำแหน่งปัจจุบัน *(มีแล้ว — ยืนยัน)*
- [ ] หยุด / เล่นต่อ ได้
- [ ] โหมดนี้ไม่มีปุ่มบันทึก/แก้ (view อย่างเดียว ตาม contract WT-0)

## นอกขอบเขต
- การแก้โน้ต/เนื้อ (โหมดแก้ = WT-D)
