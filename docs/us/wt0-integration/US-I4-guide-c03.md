## User Story
**ในฐานะ** คนทำเพลงภายนอก (ไม่มีบัญชี)
**ฉันต้องการ** อ่านในคู่มือว่าจะเสนอเพลงเข้าคลังยังไง
**เพื่อ** ทำเพลง → ดาวน์โหลด JSON → ส่งอีเมลให้ทีมได้ถูกขั้นตอน

## Acceptance Criteria
- [ ] เพิ่มขั้นตอนใน `Guide.vue` (หัวข้อ "ห้องทำเพลง" ~บรรทัด 227) ตามร่าง:
  1. คีย์เพลงให้เสร็จ
  2. กด "ดาวน์โหลด JSON" (สำเนางานของคุณ · เว็บไม่บันทึก · เปิดกลับมาแก้ได้ด้วย "อัปโหลด JSON")
  3. ส่งอีเมลถึงทีม **พร้อมแนบไฟล์ JSON**
  4. ทีมตรวจแล้วนำขึ้นคลัง
- [ ] อีเมลทีมที่จะใส่ = **รอ P'Aim ให้** (ยังไม่ hardcode มั่ว)
