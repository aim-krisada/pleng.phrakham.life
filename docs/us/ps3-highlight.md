## H1 — ไฮไลต์ไล่ทีละโน้ต/พยางค์
ในฐานะคนร้องตาม ฉันอยากให้ไฮไลต์ไล่ทีละคำ/โน้ต ตรงจังหวะ ไม่ใช่ทั้งช่องสว่างพร้อมกัน
**AC:**
- เล่น → ไฮไลต์เดิน**ทีละโน้ต** (คำ+โน้ตปัจจุบันเด่น) ไม่ใช่ระดับ segment (ทั้งช่องค้าง)
- **แตะบรรทัด/โน้ต = กระโดดไปเริ่มตรงนั้น**
- **v2** (1 พยางค์/โน้ต): ไฮไลต์รายพยางค์ · **v1** (เอื้อน 1 คำ/หลายโน้ต): fallback ไฮไลต์คำค้างข้ามโน้ตที่ลาก (per-note index แต่คำเดียว)
- **อย่าแก้ "จังหวะเพี้ยน" แยก** — งานนี้เขียนตัวจับจังหวะ/index ใหม่อยู่แล้ว
