# US-B01 — ดูแผ่นเพลงเต็มพร้อมพิมพ์

**Worktree:** WT-B · **Branch:** `wt-b-print` · **คู่ DS:** `ds/wt-b-print/DS-B01-view-full-sheet.md`
**สถานะ:** spec (รอ dev)
**โยง mission:** คนทั่วไปพิมพ์แผ่นเพลงได้เอง

## User Story
**ในฐานะ** คนที่อยากได้แผ่นเพลงไปใช้
**ฉันต้องการ** เห็นแผ่นเพลงทั้งเพลง (โน้ตตัวเลข + คอร์ด + เนื้อ) แบบเดียวกับที่จะพิมพ์ออกมา
**เพื่อ** ตรวจก่อนพิมพ์ว่าครบและถูกต้อง

## Acceptance Criteria
- [ ] แสดงทั้งเพลงเรียงตามลำดับการร้อง *(มีแล้ว)*
- [ ] มีโน้ต + คอร์ด + เนื้อ ครบ
- [ ] โหมดนี้อ่านอย่างเดียว (ตาม contract WT-0)
