# US-B02 — พิมพ์ A4 สวย ไม่ตกขอบ

**Worktree:** WT-B · **Branch:** `wt-b-print` · **คู่ DS:** `ds/wt-b-print/DS-B02-print-a4-clean.md`
**สถานะ:** spec (รอ dev)
**โยง mission:** ได้แผ่นเพลงพร้อมใช้แจกในคริสตจักร

## User Story
**ในฐานะ** คนที่จะพิมพ์แผ่นเพลง
**ฉันต้องการ** สั่งพิมพ์เป็น A4 ที่จัดหน้าสวย ไม่มีอะไรตกขอบหรือโดนตัดกลางท่อน และเลือกคีย์ที่จะพิมพ์ได้
**เพื่อ** ได้แผ่นเพลงพร้อมใช้จริง

## Acceptance Criteria
- [ ] print แล้วได้ A4 จัดหน้าเรียบร้อย ไม่ตกขอบ *(มีแล้ว/ปรับ)*
- [ ] ไม่ตัดกลางท่อน (page-break คุมได้)
- [ ] เลือกคีย์ก่อนพิมพ์ได้ (transpose)
- [ ] ตอนพิมพ์ซ่อน UI ที่ไม่เกี่ยว เหลือแต่แผ่นเพลง

## นอกขอบเขต
- export PDF ผ่านเซิร์ฟเวอร์ (ใช้ print-to-PDF ของเบราว์เซอร์ก่อน)
