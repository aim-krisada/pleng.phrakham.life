# US — ps3: รองรับโน้ตจุดคู่ (double-dot)

Backlog: B027 · DS: `docs/ds/ps3-double-dot.md`

## ที่มา
ตอนนี้แอปรองรับ "โน้ตประจุด" จุดเดียว (`5.` = ยาว ×1.5) แต่ยังไม่รองรับ **จุดคู่** (`5..` = ยาว ×1.75) ซึ่งเป็นมาตรฐาน jianpu (Wikipedia: จุดเดียว +½ · สองจุด +¾) · P'Aim 2026-07-08 เลือกเพิ่ม**จุดคู่ก่อน** (ตัวอื่น—โน้ต 32/คอร์ดซ้อน/grace/glissando—พักไว้ ดูเหตุผลใน chat/บันทึก)

## DD1 — พิมพ์จุดคู่แล้วได้ความยาวถูก
ในฐานะคนคีย์เพลง ฉันอยากพิมพ์จุดคู่หลังโน้ตแล้วได้เสียงยาว ×1.75 ตามมาตรฐาน
**AC:**
- พิมพ์ `5..` (สองจุดหลังโน้ต) → โน้ตยาว **×1.75** (จุดเดียว `5.` = ×1.5 เท่าเดิม)
- **นับจังหวะในห้อง** (ตัวเลข x/y) คิดจุดคู่ถูก
- **กดฟังทำนอง** เสียงยาวตามจุดคู่
- **แผ่นเพลง + โหมดแก้ไข แสดง 2 จุด** ต่อท้ายโน้ต
- ของเดิมไม่พัง: จุดเดียว `5.` · จุดล่างอ็อกเทฟ `.5` · โน้ตติดกัน `5..3` ยังทำงานเหมือนเดิม
- **คู่มือ** เพิ่มแถวอธิบายจุดคู่
