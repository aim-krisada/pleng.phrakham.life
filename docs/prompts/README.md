# Prompts — ตัวเปิด session ตามบทบาท

ก๊อป/ชี้ prompt ตามบทบาทที่จะเปิด session:
- **`sa.md`** — เปิด session เป็น **SA** (คุยกับพี่เอม · แปลง requirement → US/DS · ประสาน dev · ตัดสิน merge)
- **`dev.md`** — เปิด session เป็น **dev** (สร้างโค้ดตาม US/DS ใน worktree ของตัวเอง)

**พี่เอมใช้:** บอก session ว่า
- `อ่าน docs/prompts/sa.md · ผมต้องการ ...` → session นั้นกลายเป็น SA แล้วรับงาน
- `อ่าน docs/prompts/dev.md · ผมต้องการ ทำ WT-A ...` → session นั้นกลายเป็น dev แล้วลงมือ
