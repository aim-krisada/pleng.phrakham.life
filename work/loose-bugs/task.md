# บั๊กเล็กค้าง (ยังไม่จัดคิว · ไม่บล็อก)

แต่ละอันเล็กพอทำจบใน 1 session · หยิบทีละอัน · ทำเสร็จอันไหนลบบรรทัดนั้น · หมดแล้วลบ folder

- **B117 — ถามพี่เปา 1 คำถาม** (ยังไม่ใช่งานโค้ด): โน้ตชุดเร็ว `4 3 2 1` ที่แต่ละตัวมีคำ อยากให้มัดคานติดกันไหม — คลังมัดแค่ **10.9%** ถ้ามัดตามจังหวะได้ **68.7%** · ⛔ **ไม่ใช่บั๊ก มี golden test ที่พี่เปาเคยขอ ห้ามเปลี่ยนเองโดยไม่ถาม** ([[closed/beam-per-syllable]])
- **B115** — กด back จากหน้าเพลง เด้งกลับชั้นหนังสือ แทนกลับรายการในเล่ม (pre-existing)
- **B108 quirk** — เลือก "— ไม่ระบุธีม —" แล้วเผยแพร่ ธีมเก่ากลับมา → ต้องทำ tri-state
- **ชิป fermata** ทับแป้นตอนโน้ตอยู่ล่างสุดจอเตี้ย (ไม่ใช่ regression)
- **`𝄐` glyph บนมือถือ** ยังไม่ยืนยันว่าขึ้นเป็นกล่อง □ ไหม · ถ้าเป็น → เปลี่ยนเป็น SVG (**กระทบของที่ live**)
