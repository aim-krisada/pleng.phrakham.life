# Melisma-slur review — "หางทำนองท้ายวรรค" (for P'Aim / พี่เปา)

Derived melisma slurs (P2) draw a curve over a worded note + following blank "attack" notes.
The 5 crops below are the LONG-run cases from a full-library sweep (149 songs, 1,358 derived
slurs; 98% are the normal 2-note word+blank). Each is a **last sung word held over a wordless
melodic tail** — arguably a real เอื้อน, arguably over-draw. Human ear decides.

Rendered live from Supabase content via Chromium (headless + CDP), commit `8c508be`.
