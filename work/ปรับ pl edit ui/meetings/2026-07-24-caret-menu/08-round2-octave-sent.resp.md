ยินดีต้อนรับสู่รอบ 2 ครับ! สำหรับการพิมพ์เพื่อระบุ ระดับเสียง (Octave) ด้วยคีย์บอร์ดล้วนๆ โดยไม่ต้องพึ่งกราฟิกบรรทัด 5 เส้นนั้น มีมาตรฐานที่นิยมใช้กันอยู่ 3-4 ระบบหลักๆ ขึ้นอยู่กับจุดประสงค์การใช้งาน ดังนี้ครับ

1. ระบบ Scientific Pitch Notation (SPN)
ระบบนี้เป็นสากล เข้าใจง่าย และพิมพ์สะดวกที่สุด โดยจะใช้ ชื่อโน้ต + ตัวเลข เพื่อระบุช่วงเสียงอย่างชัดเจน

C4 หมายถึง Middle C (โดกลาง)

หากต้องการเสียงสูงขึ้น 1 Octave จะเพิ่มตัวเลขเข้าไป เช่น C5, C6

หากต้องการเสียงต่ำลง จะลดตัวเลขลง เช่น C3, C2

การใช้งาน: เหมาะสำหรับการสื่อสารทฤษฎีดนตรีทั่วไป หรือจดโน้ตแบบเร็วๆ

2. ระบบ ABC Notation และ LilyPond
ระบบนี้มักใช้ในการพิมพ์ข้อความ (Text) เพื่อให้โปรแกรมแปลงไปเป็นชีทโน้ตเพลงบรรทัด 5 เส้น โดยจะใช้ ตัวพิมพ์เล็ก/ใหญ่ + สัญลักษณ์เครื่องหมายวรรคตอน

C (ตัวพิมพ์ใหญ่) = เสียงต่ำ (Bass)

c (ตัวพิมพ์เล็ก) = เสียงกลาง (Middle)

c' (ตัวพิมพ์เล็ก + เครื่องหมาย Apostrophe) = กระโดดขึ้น 1 Octave (เสียงสูง)

C, (ตัวพิมพ์ใหญ่ + เครื่องหมาย Comma) = กระโดดลง 1 Octave (เสียงต่ำมาก)

การใช้งาน: เหมาะสำหรับคนที่เขียนโค้ดเพื่อสร้างชีทโน้ตเพลง

3. ระบบ MML (Music Macro Language) / Chiptune
ระบบนี้ใช้สำหรับเขียนคำสั่งเสียง หรือทำเพลงแนว 8-bit ในอดีต มักใช้การตั้งค่าค่า Octave ไว้ล่วงหน้า หรือใช้เครื่องหมายทางคณิตศาสตร์

ใช้คำสั่ง O (โอ) ตามด้วยเลข เช่น O4 C D E (แปลว่าให้โน้ต C D E เล่นในระดับเสียง Octave ที่ 4 ทั้งหมด)

ใช้เครื่องหมาย มากกว่า/น้อยกว่า (>, <) เพื่อเลื่อน Octave ขึ้นลงแบบฉับพลัน เช่น C > C (เล่น C ตัวแรก แล้วกระโดดขึ้นไปเล่น C ตัวที่สูงกว่า 1 Octave)

การใช้งาน: โปรแกรมจำพวก Tracker หรือการเขียนสคริปต์ทำเพลงเกมเก่าๆ

4. การใช้ Caret (^) และการจดแบบอิสระ (Shorthand)
สืบเนื่องจากรอบที่แล้ว หากคุณแค่ต้องการจดโน้ตไว้อ่านเองหรือเล่นดนตรีเอง การนำเครื่องหมายที่มีบนคีย์บอร์ดมาใช้ให้เห็นภาพง่ายๆ ก็เป็นที่นิยมครับ

^C หรือ C^ = ดันเสียง C ขึ้น 1 Octave (ใช้ Caret ชี้ขึ้น)

vC หรือ C_ = ดึงเสียง C ลง 1 Octave (ใช้ตัววีเล็ก หรือ Underscore)

คุณกำลังหาฟอร์แมตสำหรับการ "จดโน้ตไว้อ่านเอง/ร้องเอง" หรือว่ากำลังต้องพิมพ์คำสั่งลงใน "โปรแกรมซอฟต์แวร์เฉพาะทาง" อยู่ครับ? ผมจะได้แนะนำเจาะจงให้ตรงกับการนำไปใช้มากยิ่งขึ้นครับ
