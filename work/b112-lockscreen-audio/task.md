# B112 — เล่นเพลงต่อตอนปิดจอ + คุมจากจอล็อก

**Objective:** P'Aim ยืนยันเอง + ส่งรูป — จอล็อกโชว์ ชื่อ + ปกเพลง + แถบเวลา + ก่อนหน้า/เล่น/ถัดไป · เล่นต่อเมื่อปิดจอ · **P'Aim สั่ง "ให้เสร็จทีละงาน" = คิวถัดไป**

**🔴 จุดตาย (ห้ามรับปากว่าง่ายก่อนสำรวจ):**
- ยืม lib กลาง `assets/pk-audio.js` ของพระคำ (UMD `window.PKAudio`) — **พระคำเป็นเจ้าของ ขอผ่าน pk pm เท่านั้น ห้ามแก้ตรง** ([[pair-sop §7]])
- **ไม่ใช่ drop-in:** พระคำ = mp3 อัดไว้ล่วงหน้า · เรา = สังเคราะห์สดตอนกด (`src/lib/audioExport.js`) + ไฮไลต์ index-based ไม่ใช่ time-based → ต้องเลือก: ป้อน blob+timing เอง **หรือ** ตัดไฮไลต์ เอาแค่ background + lock screen
- MediaSession ไม่ต้อง https (LAN ได้) · ต้องเสิร์ฟด้วย server รองรับ **Range** (`py -m http.server` ไม่รองรับ → แถบเลื่อนเด้ง 0)
- **ต้องนัดหู P'Aim ไว้ก่อนเริ่ม** — เสียง/จังหวะเล่นต่อ = ชั้น `audio` AI เซ็นไม่ได้

**รายละเอียด:** อ่าน `pm/design-audio-mp3.md` + `pm/report-audio-player.md` (repo พระคำ) ก่อน scope · เดิมอยู่ `HANDOFF-pm35.md §4`
